# Design Differentiation Task

## Purpose

LP_Creator_Team must avoid one-pattern AI-like design outputs. The design must be selected according to the message, target user, offer, emotional CVR goal, and conversion intent.

## Required External Design References

Use these as design reference sources when planning visual direction. Do not copy blindly. Extract principles and patterns.

- Icons: https://pa-tu.work/tools/icon-search/
- UI Design Dictionary: https://ui-design-dictionary.pages.dev/
- Material Design Components: https://m3.material.io/components
- Curated Web Design References: https://craftwork.design/curated/websites/

## Mandatory Design Thinking

For every LP, design agents must decide:

1. Icon style: line, filled, duotone, pictogram, illustration, or none
2. UI pattern: cards, tabs, accordions, stepper, comparison table, pricing table, sticky CTA, modal, forms
3. Typography: trustworthy, energetic, premium, friendly, technical, editorial, or recruitment-oriented
4. Motion: none, micro-interaction, scroll reveal, CTA emphasis, FAQ animation, process animation
5. Visual hierarchy: what should be seen first, second, and third
6. Emotional role: clarity, expectation, trust, urgency, reassurance, excitement
7. Conversion role: reduce anxiety, increase motivation, explain mechanism, support proof, or push CTA

## Anti-Pattern

Reject designs that are:

- Generic SaaS blue gradient hero
- Random icons without semantic role
- Same card layout repeated everywhere
- Motion added only for decoration
- Font choice with no message intent
- AI-like abstract illustrations
- Templates that could fit any company
