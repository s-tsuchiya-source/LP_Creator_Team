# Action Motivation CTA Patterns

## Role in LP

CV直前の行動意欲を高めるCTAパターン。

## Best Use Cases

- 最終CTA セクション
- 中盤の主要CTA
- 限定性提示時

## Required Elements

- ハードル低い行動を主CTA
- 「今動く理由」
- ベネフィット再訴求
- 安心要素（解約・返金 等）
- CV後の流れ予告

## High Conversion Pattern

### 型A: ハードル段階別CTA

```markdown
[3分で資料を受け取る]   主CTA（最低ハードル）
[無料デモを予約する]    副CTA（中ハードル）

★ 14日間無料トライアル / クレジットカード不要
★ いつでも解約可能
```

### 型B: ベネフィット再訴求 + CTA

```markdown
## [サービス名] で、採用業務を変えませんか？

✓ 採用工数を最大60%削減（自社調べ・100社）
✓ 応募者対応の質を向上
✓ 経営層への報告が自動化

導入企業1,200社が選ぶ採用管理ツール

[今すぐ無料で資料請求]   [無料デモを予約]

★ 14日間無料トライアル / クレジットカード不要
★ 月単位の解約OK
★ 専任担当による導入支援

※ 効果には個人差・業種差があります
```

### 型C: 「今動く理由」型

```markdown
## なぜ、いま検討するのが良いか

### 1. 採用シーズンへの備え
新年度の採用活動が始まる前に、業務フローを整えませんか？

### 2. 限定特典（実態に即す）
2026年X月X日まで：
- 初期設定支援が無料
- データ移行代行が無料

### 3. 14日間の無料トライアル
今申込めば、X月X日までトライアル可能

[今すぐ無料で資料請求]

※ 期限は実態に即した正直な期日です
```

### 型D: CV後の流れ予告型

```markdown
## 資料請求後、こんな価値があります

📊 **すぐに（即時）**
資料がメールで届きます。
- 機能一覧
- 同業種事例 5社
- 価格・プラン詳細
- ROI試算シート
- セキュリティ情報

💬 **任意で（ご希望時）**
個別相談で貴社課題を整理（30分・オンライン）

🎯 **無料で（14日間）**
無料トライアルで実環境テスト
（クレカ不要・自由解約）

🚀 **そして**
業務改善の道筋が見える

[3分で資料を受け取る]
```

### 型E: 不安先回り解消型

```markdown
## ご検討中の方へ

「資料請求するだけで、しつこく営業されないか心配」
そう思う方が多くいらっしゃいます。

ご安心ください：

✓ 営業からの強引な連絡はありません
✓ 個別相談はご希望の方のみ
✓ メルマガ登録は任意
✓ 解除はいつでも可能

「資料を見るだけ」のつもりでお気軽に。

[3分で資料を受け取る]

※ プライバシーポリシー: [リンク]
```

## Bad Pattern

### NG型: CTA1つだけ

```
[お問い合わせ]
```

→ ハードル選択肢なし。離脱率高。

### NG型: 「お問い合わせ」だけのCTA

```
[お問い合わせ]
[詳細はこちら]
```

→ ベネフィット欠如。CV理由不明。

### NG型: 嘘の限定性

```
本日限定！
今だけ特別価格！
※ 実際は常時セール
```

→ 景表法違反。信頼毀損。

### NG型: CV後の流れ不明

```
[今すぐ申し込む]
（申込後の流れの説明なし）
```

→ 不安で離脱。

## Copy Examples

### CTA文言（ハードル低）
- 「3分で資料を受け取る」
- 「メールアドレスだけで資料DL」
- 「無料で診断する」
- 「カジュアル面談を予約する」

### CTA文言（ハードル中）
- 「無料デモを予約する」
- 「個別相談（無料）に申し込む」
- 「14日間無料で試す（クレカ不要）」

### CTA文言（ハードル高）
- 「契約手続きへ進む」
- 「今すぐ申し込む」
- 「購入する」

### 安心要素テキスト
- 「クレジットカード不要」
- 「いつでも解約可能」
- 「営業からの強引な連絡なし」
- 「個別相談はご希望の方のみ」

## Design Notes

- 主CTAは大きく目立つ色
- 副CTAはトーン落ち（線・ゴースト）
- 安心要素はCTA直下に小さめ
- 限定性は控えめに（嘘でない範囲）

## Measurement Notes

- 主CTA / 副CTA のクリック率
- 「今動く理由」セクションの閲覧時間
- 「資料請求後の流れ」のスクロール深度

## Checklist

- [ ] ハードル段階別CTA（最低2個）
- [ ] ベネフィット再訴求
- [ ] 「今動く理由」（実態に即す）
- [ ] CV後の流れ予告
- [ ] 営業されすぎ不安への対応
- [ ] 嘘の限定性なし
