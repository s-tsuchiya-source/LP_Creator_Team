# Anxiety to Confidence Patterns

## Role in LP

不安を安心・期待に変える表現パターン。CV直前の心理ハードルを下げる。

## Best Use Cases

- FAQ セクション
- 価格セクション
- 最終CTA直前
- フォーム周辺

## Required Elements

- 6カテゴリ不安への対応
- 「共感 → 論拠 → 証拠」構造
- 法的に問題のある保証なし
- 不利情報の正直な記載

## High Conversion Pattern

### 型A: 「契約後も自由に判断できる」設計

```markdown
## ご検討中の方へ

「いきなり契約するのは不安」という声をよくいただきます。

[サービス名] では、以下の設計で安心していただけます：

✓ 14日間の無料トライアル（クレカ不要）
✓ 月単位の解約OK（最低利用期間なし）
✓ データはいつでもCSVエクスポート可能
✓ 契約後の伴走支援（3ヶ月）

「契約後も、自由に判断してください」が私たちの方針です。

実際、トライアル後の継続率は85%（自社調べ・2025年実績）。
無理な引き留めは一切いたしません。
```

### 型B: 6カテゴリ不安の一覧解消

```markdown
## よくある不安と、私たちの答え

### 価格が高そう
ROI試算では、月50時間の工数削減で月15万円相当の効果。
月額9,800円との差額 = 月14万円のリターン。

### 導入に時間がかかりそう
最短2週間で導入完了。専任担当が支援します。
データ移行は無料で代行可能。

### 効果が出るか不安
同業種の導入事例を、無料資料に収録しています。
A社（60%削減）、B社（応募数1.5倍）など。

### 運用が大変そう
研修・チャットサポート・FAQ動画を完備。
導入企業の85%が「想定より簡単」と回答。

### 社内説明が不安
ROI試算 / 比較表 / 経営層向け1pサマリー を無料提供。
社内会議でそのまま使えます。

### 失敗したくない
月単位解約OK / 14日間無料 / データエクスポート可。
「契約後も自由」が私たちの設計です。
```

### 型C: 営業されすぎ不安への対応

```markdown
## 資料請求後の対応について

✓ 営業からの強引な連絡はありません
✓ ご希望の方のみ、個別相談を実施
✓ メルマガ登録は任意
✓ 解除はいつでも可能

資料を見るだけ、でもOKです。

ご不明点があれば、メールでお問い合わせください。
お電話はかけません。
```

### 型D: 失敗不安への対応

```markdown
## 万が一、合わなかった場合

[サービス名] では、以下の安全装置を用意しています：

### 解約自由
- 月単位の解約OK
- 最低利用期間なし
- 違約金なし

### データポータビリティ
- いつでもCSVエクスポート可能
- API経由でのデータ取り出しも可
- 他社システムへの移行支援あり

### 期間限定保証
- 14日間無料トライアル
- 不満な場合の返金（条件付き）

「導入してから後戻りできない」を絶対に避ける設計です。
```

### 型E: 上司説明不安への対応

```markdown
## 上司への説明が不安な方へ

社内検討用の資料を、無料でご用意しています：

### 経営層向け
- 1pサマリー（A4 1枚）
- ROI試算シート（あなたの会社規模）
- セキュリティ要件書

### 部門責任者向け
- 機能比較表（競合A・B社 vs 自社）
- 同業種の導入事例集
- 業務フロー比較

### 情シス向け
- セキュリティチェックリスト
- API連携仕様書
- 認証取得証明書

社内会議で、そのままお使いいただけます。

[資料パックを受け取る（無料）]
```

## Bad Pattern

### NG型: 「ご安心ください」だけ

```
ご安心ください。
万全のサポート体制でご支援します。
業界最高の品質をお届けします。
```

→ 具体性なし。安心できない。

### NG型: 法的に問題のある保証

```
絶対に成功します
100%の効果保証
失敗の心配は一切なし
```

→ 法務リスク。長期的に信頼毀損。

### NG型: 不利情報を隠す

```
（解約条件・最低利用期間を一切書かない）
（返金条件を曖昧にする）
```

→ 後でトラブル。信頼毀損。

## Copy Examples

### 安心を作る表現
- 「契約後も、自由に判断してください」
- 「無理な引き留めは一切いたしません」
- 「営業からの強引な連絡はありません」
- 「万が一、合わなかった場合は…」

### 避けるべき表現
- 「絶対」「100%」「必ず」
- 「ご安心ください」だけ
- 「業界最高の品質」（根拠なし）

## Design Notes

- 不安カテゴリごとにアイコン
- 「✓」で安全装置を視覚化
- ストーリー型（「Aさんの場合」）も有効
