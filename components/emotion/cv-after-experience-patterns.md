# CV After Experience Patterns

## Role in LP

CV後にユーザーが得る体験を明示し、「申込後の不安」を解消する。
サンクスページ・自動返信メール・次の流れを設計する。

## Best Use Cases

- 最終CTA直前のセクション
- サンクスページ（CV後）
- 自動返信メール

## Required Elements

- 申込直後の流れ
- 短期（数日以内）の対応
- 中期（1〜2週間）の対応
- 営業対応の方針
- メルマガ・連絡方針

## High Conversion Pattern

### 型A: 流れ予告型（最終CTA直前に配置）

```markdown
## 資料請求後の流れ

### Step 1: 即時
ご記入のメールアドレスに、資料が届きます

### Step 2: 当日〜翌営業日
担当者からのご挨拶メールをお送りします
（営業電話はかけません）

### Step 3: ご希望時
個別相談・デモのご予約が可能です
（任意）

### Step 4: 14日間
無料トライアルで実環境テスト
（クレカ不要・自由解約）

「資料を見るだけ」でも、もちろんOKです。

[3分で資料を受け取る]
```

### 型B: サンクスページ型

```markdown
# 資料請求ありがとうございます

ご記入いただいたメールアドレスに、
資料を送信いたしました。

## 次の流れ

### すぐに
- ✉️ 資料が届きます
- （迷惑メールフォルダもご確認ください）

### 3営業日以内
- 担当者からご挨拶メールをお送りします
- 営業電話はかけません

### ご希望時
- 個別相談（30分・オンライン）
- 無料デモ
- 14日間無料トライアル

## よくあるご質問

**Q. 資料が届かない場合は？**
迷惑メールフォルダをご確認ください。
3時間以上経っても届かない場合は、こちらまで：[メールアドレス]

**Q. 営業電話はありますか？**
かけません。
ご希望の方のみ、メールで個別相談をご案内します。

**Q. デモを予約したいのですが？**
こちらから24時間予約可能：[予約URL]

## 関連資料

- [競合比較表 PDF]
- [ROI試算シート]
- [導入事例集]

## SNS でも情報発信中

- Twitter: [@xxx]
- LinkedIn: [URL]
```

## 14日間 無料トライアル

クレジットカード不要で、すぐに開始できます：

[サービス名] 担当 [氏名]
[会社名]

### 型D: 採用LP用 サンクスページ

```markdown
# カジュアル面談のご予約、ありがとうございます

## ご予約内容

- 日時: [予約日時]
- 形式: オンライン
- 担当: [担当者名]

## 当日までの流れ

### 1〜2営業日以内
担当者よりご確認メールをお送りします。
ZoomのURLもお送りします。

### 当日（30分）
- 自己紹介・お話のきっかけ
- 仕事内容・カルチャーの説明
- ご質問にお答えします
- カジュアルな雑談形式です

### 後日
- 興味があれば、エントリーへ
- 興味がなくても、お断りでOKです

## ご準備不要です

- 履歴書不要
- 事前資料不要
- 服装自由（オンラインなのでカジュアルでOK）

## ご質問・キャンセル

[連絡先メールアドレス]

## 採用情報

- 採用ピッチ動画: [URL]
- 社員インタビュー: [URL]
- メンバー紹介: [URL]
```

## Bad Pattern

### NG型: サンクスページが汎用的すぎる

```
お申し込みありがとうございます。
担当者よりご連絡いたします。
```

→ 不安が残る。

### NG型: 流れの予告なし

```
[今すぐ申し込む]
（申込後の流れの説明が一切ない）
```

→ 「申込後何が起きる？」で離脱。

### NG型: 営業電話の有無が不明

```
（営業電話があるのか、ないのか書かない）
```

→ 不安で離脱。

## Copy Examples

### 「営業されすぎない」を伝える表現
- 「営業電話はかけません」
- 「ご希望の方のみ、個別相談を実施」
- 「メルマガ登録は任意」
- 「いつでも配信解除可能」

### CV後の安心を作る表現
- 「3営業日以内に担当者よりメール」
- 「質問はメールで返信ください」
- 「資料を見るだけ、でもOKです」

## Design Notes

- サンクスページはシンプルに
- 次のアクションを明確に
- SNSや関連資料へのリンクも配置

## Measurement Notes

- サンクスページのCV計測
- 関連資料・トライアル開始のクリック率
