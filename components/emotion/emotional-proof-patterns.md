# Emotional Proof Patterns

## Role in LP

感情と信頼をつなぐ証拠表現。数字の証拠だけでなく、感情に届く証拠を作る。

## Best Use Cases

- 実績セクション
- お客様の声
- 事例セクション
- 認証バッジ

## Required Elements

- 数値証拠（論理）
- ストーリー証拠（感情）
- 第三者証拠（権威）
- ペルソナ近接事例（共感）

## High Conversion Pattern

### 型A: ストーリー型お客様の声

```markdown
## A社（製造業 / 100名）の場合

【導入前】
人事部長は、毎週月曜日の朝が憂鬱でした。
スプレッドシートで応募者管理、抜け漏れが頻発。
「あの応募者、返信した？」と上司に聞かれるたびにヒヤヒヤ。

【導入のきっかけ】
営業からの強引な連絡がない、という資料の説明に安心。
専任担当による導入支援も決め手でした。

【導入3ヶ月後】
月150時間 → 月60時間（60%削減）。
朝、ダッシュボードを見るだけで状況把握。
上司から「楽になったね」と評価される日々。

「あの時、資料を請求して本当によかった」

— 人事部長 山田様（仮名・許諾済）
```

### 型B: 数値 + 感情のセット

```markdown
## 導入企業の声

### 「もう手作業で疲弊しない」
業務工数: 月150時間 → 月60時間（60%削減）
- B社 採用担当 田中様（許諾済）

### 「経営層からの信頼が変わった」
報告自動化: 週8時間 → ほぼゼロ
- C社 人事部長 佐藤様（許諾済）

### 「採用の質が見えるようになった」
データ可視化: 経路別CV率がリアルタイム把握
- D社 経営者 鈴木様（許諾済）

※ 自社調べ・2025年実績。効果には個人差があります
```

### 型C: 同業種・同規模の安心感

```markdown
## あなたと近い企業の事例

### 製造業 / 50〜100名
- A社: 採用工数 60%削減
- B社: 内定辞退率 30%減

### IT・SaaS / 100〜300名
- C社: 応募数 1.5倍
- D社: 採用部門の満足度 90%

### 小売 / 30〜80名
- E社: 採用単価 40%減

※ あなたの業界の事例集を、無料資料に収録しています
```

### 型D: 失敗回避エピソード

```markdown
## なぜ85%の企業が継続利用するのか

【F社の場合】
他社サービスから乗り換え。

【乗り換え理由】
- 「営業の押し売りがない」
- 「専任担当がしっかり伴走してくれた」
- 「現場が想像以上に早く定着した」

導入から3年経っても継続利用中。
「もう他社には戻れない」（F社人事部長 高橋様 / 許諾済）

※ 自社調べ・2025年実績
```

### 型E: 第三者の権威証拠

```markdown
## 第三者からの評価

### 受賞
- [賞名] 受賞（2025年・[主催団体]）

### 認証
- ISO 27001 取得
- Pマーク取得

### メディア掲載
- [媒体名]（2025年X月号）「注目のSaaS 10社」
- [媒体名]（2025年X月）「採用DXの最新動向」

### 専門家推薦
「中小企業の採用業務に最適化された設計」
- 人事コンサルタント [氏名] 様
```

## Bad Pattern

### NG型: 数字だけの実績

```
- 1,200社導入
- 顧客満足度No.1
- 業界シェア1位
```

→ 数字は大きいが感情に届かない。

### NG型: テンプレお客様の声

```
「最高のサービスです！」
「すごく便利です！」
「使いやすいです！」
```

→ 信頼性がない。

### NG型: 出典なき推薦

```
「業界でも最高評価」
「専門家からも絶賛」
```

→ 法務リスク。

## Copy Examples

### 感情に届く表現
- 「もう手作業で疲弊しない」（解放感）
- 「上司から評価される日々」（達成感）
- 「失敗のリスクが見えなくなった」（安心感）
- 「あの時、決断してよかった」（後悔のなさ）

### 避けるべき表現
- 「最高のサービス」（抽象）
- 「絶対におすすめ」（押し付け）
- 「業界一」（根拠なし）

## Design Notes

- 顔写真（許諾済）を大きめに
- ストーリーは読みやすい段組
- 数値はバッジ風
- 認証ロゴは整列

## Measurement Notes

- 各事例の閲覧時間
- 「もっと見る」「他の事例」のクリック計測

## Legal Notes

- 顧客の許諾必須
- イニシャル + 業種・役職で個人特定回避
- 「効果には個人差があります」明記

## Checklist

- [ ] ストーリー型のお客様の声を最低1件
- [ ] 数値 + 感情のセット
- [ ] 同業種・同規模事例
- [ ] 第三者証拠（受賞 / 認証 / メディア）
- [ ] 顧客許諾済み
- [ ] 「個人差」明記
