# Expectation Copy Patterns

## Role in LP

期待感を生むコピーパターン。誇張ではなく根拠で期待を作る。

## Best Use Cases

- FV のサブコピー
- 解決策セクション
- 実績セクション
- CTA直前

## Required Elements

- 数値根拠（出典付き）
- 事例根拠
- プロセス根拠
- 「個人差」明記
- 誇大表現排除

## High Conversion Pattern

### 型A: 数値根拠型

```markdown
導入企業1,200社が選ぶ採用管理ツール
（自社調べ・2026年4月時点）

平均60%の業務工数削減を実現
（100社平均・効果には個人差があります）
```

### 型B: 事例根拠型

```markdown
あなたの業界で、すでにこんな成果が出ています：

- A社（製造業 / 100名）: 採用工数 60%削減
- B社（IT / 200名）: 内定辞退率 30%減
- C社（小売 / 50名）: 月間応募数 1.5倍

※ 自社調べ・2025年実績。効果には個人差があります
```

### 型C: プロセス根拠型

```markdown
導入から成果まで：

【最短2週間】導入完了
- 専任担当が初期設定を支援（無料）
- データ移行は自動 / または代行（無料）

【1ヶ月後】初期効果を実感
- 操作に慣れ、20%の工数削減

【3ヶ月後】本格的な効果
- 60%の工数削減（自社調べ）

「現場が定着するまで支援」が私たちの約束です。
```

### 型D: 第三者根拠型

```markdown
[業界誌名] にて「2025年注目のSaaS」に選出
ISO 27001 / Pマーク 取得済み
[賞名] 受賞（2025年・[主催団体]）
```

### 型E: 組み合わせ型（最強）

```markdown
## 採用業務を、最大60%削減

導入企業1,200社の平均（自社調べ・100社）

✓ 同業種事例 5社（資料に収録）
✓ 最短2週間で導入完了
✓ 専任担当が伴走（3ヶ月）
✓ ISO27001 / Pマーク取得済み

※ 効果には個人差・業種差があります

[3分で資料を受け取る]
```

## Bad Pattern

### NG型: 誇大表現

```
業界の常識を覆します
絶対に成果が出ます
100%の効果を保証
劇的に変わります
業界No.1（根拠なし）
```

→ 信頼を失う / 法務リスク

### NG型: 抽象的期待

```
あなたの業務が劇的に変わります
業界をリードする最高のツール
革新的なソリューション
```

→ 何も伝わらない

### NG型: 出典なし数値

```
業務効率が80%アップ
顧客満足度No.1
成長率20%
```

→ 景表法リスク

## Copy Examples

### CV直前で期待を最大化

```markdown
## あなたの3ヶ月後を、想像してみてください

採用業務に追われる毎日が、
ダッシュボードでサクッと確認できる業務に変わる。

上司から「分かりやすい」と評価される。
残業せずに帰れる金曜日。

そんな未来を、無料の資料で具体的にイメージしてください。

資料には、同業種事例5社・ROI試算が含まれます。
営業からの強引な連絡はありません。

[3分で資料を受け取る]

※ 効果には個人差があります
```

## Design Notes

- 数値は太字 or バッジ
- 「※個人差」は小さめのフォント（ただし読める）
- 出典は表示
- 過剰なグラフィックは避ける

## Measurement Notes

- 期待感を作るセクションのスクロール深度
- CTA直前の閲覧時間
