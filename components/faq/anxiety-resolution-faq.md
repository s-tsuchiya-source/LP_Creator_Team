# Anxiety Resolution FAQ

## Role in LP

ターゲットの「最後の壁」となる不安を解消する FAQ パターン集。

## Common Anxiety Types

### 1. 価格不安
- 「高そう」「予算オーバー」
- 解消: 分割払い / ROI試算 / 無料トライアル

### 2. 効果不安
- 「本当に効くのか」「自社に合うか」
- 解消: 同業種事例 / 成果保証 / 無料診断

### 3. 時間不安
- 「導入に時間かかる」「現場に定着しない」
- 解消: 最短期間 / 専任担当 / 教育プログラム

### 4. 解約不安
- 「縛りが心配」「乗り換え失敗時のリスク」
- 解消: 月単位解約 / データエクスポート可

### 5. 信頼不安
- 「この会社大丈夫？」「サポート品質は？」
- 解消: 会社情報 / 受賞 / 認証 / サポート体制

### 6. 社内反対不安
- 「上司が反対しそう」「決裁通らない」
- 解消: 稟議書テンプレ / ROI試算 / 比較表

## Anxiety Resolution Patterns

### パターンA: 共感 + 具体解決
```
Q. 現場が新しいツールを使ってくれるか心配です

A. 多くのお客様が同じご不安をお持ちでした。
   [サービス名] では、導入時に専任担当が
   現場担当者向けの研修を実施します。

   実績として、導入3ヶ月後の現場定着率は
   平均95%（自社調査・100社）です。
```

### パターンB: 不利情報を正直に + 解決策
```
Q. 既存システムからのデータ移行は可能ですか？

A. CSV / Excel からの移行は無料で対応可能です。
   ただし、独自フォーマットの場合は別途お見積もり
   となる場合があります。

   詳細は無料相談にて、貴社の状況を確認のうえ
   お伝えします。
```

### パターンC: ROI試算で価格不安解消
```
Q. 月額9,800円は高くないですか？

A. 採用業務の工数を月50時間削減した場合、
   時給3,000円換算で月15万円の削減効果となります。

   月額9,800円との差額 = 月14万円の純利益。
   1年で約170万円の効果が期待できます。

   ※ あくまで試算値です。実際の効果は規模により変動します。
```

## Bad Pattern

- 「ご安心ください」だけ
- 「絶対」「100%」の保証
- 「失敗なし」の断定
- 不安への回避（直視しない）
