# Objection FAQ Patterns

## Role in LP

CV直前の不安・反論を解消するFAQ。離脱率を大幅に下げる。

## Required Elements

- 5〜10件
- 主要不安カテゴリ網羅
- アコーディオンUI（SP対応）

## High Conversion Pattern

### 必須カテゴリ

1. **価格・料金体系**
2. **効果・実績**
3. **導入期間・サポート**
4. **解約条件**
5. **セキュリティ・個人情報**
6. **業種・規模適合性**

### FAQ Q&A 構造

```markdown
**Q. [質問]**

[Answer]
- 共感:
- 論拠:
- 証拠:
```

## Examples（BtoB SaaS）

```markdown
**Q. 中小企業でも導入できますか？**

中小企業向けに特化した [Light プラン] をご用意しています。
5名から導入可能で、初期費用は無料です。

実際に従業員10名前後の企業様にも多数ご利用いただいています。

---

**Q. 既存システムとの連携は可能ですか？**

主要な[業界システム] とは API 連携可能です。
詳細は無料相談にてご確認いただけます。

---

**Q. 解約はいつでもできますか？**

月単位でご解約可能です。最低利用期間はありません。
ただし、初期設定費用は返金対象外となります。

---

**Q. セキュリティは大丈夫ですか？**

ISO27001 / Pマーク取得済みです。
データはAES-256で暗号化し、SSL通信を採用しています。
詳細はセキュリティホワイトペーパー（[リンク]）をご覧ください。

---

**Q. 導入までどれくらいかかりますか？**

最短2週間でご利用いただけます。
データ移行が必要な場合、専任担当が支援します。
```

## Bad Pattern

- 「ご安心ください」だけのテンプレ回答
- 反論回避のみ（具体性なし）
- 主要不安カテゴリ欠落
- 法的問題のある保証（「絶対に成功」等）
- 不利情報の隠蔽

## Design Notes

- アコーディオン形式（タップで開閉）
- 開閉アイコン明示（▼ ▲ or +/-）
- カテゴリ別グルーピング（多い場合）

## Measurement Notes

- FAQ開閉計測（どの質問が頻繁に開かれるか）
- 開閉率の高いFAQは内容の改善 or 上位配置

## Legal Notes

- 法的に問題のある保証はしない
- 「個人差があります」明記（効果系）
- リスクや制約も正直に記載

## Checklist

- [ ] 5〜10件
- [ ] 主要カテゴリ網羅
- [ ] 具体的な回答
- [ ] アコーディオンUI
- [ ] 計測実装
