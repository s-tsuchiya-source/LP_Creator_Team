# FV Component: 広告流入LP

## Role in LP

広告クリック直後の体験を提供し、メッセージマッチで離脱を防ぐ。FV単体で離脱判断されるため、最重要部品。

## Best Use Cases

- Google Ads / Meta / Yahoo 流入LP
- 検索広告連動LP
- 動画広告ランディング

## Required Elements

- 広告キーワードと連動したメインキャッチ
- 強烈なベネフィット
- メインCTA（フォーム到達）
- 補助CTA（電話 / 資料DL）
- 信頼バッジ
- ビジュアル

## High Conversion Pattern

```
[広告キーワード] を、[改善幅] で解決する [サービス名]

[広告コピーと連動したサブコピー]

[今すぐ無料で資料請求]   [電話で相談]

★[実績数値]   ★[認証]
```

実装例（広告キーワード「採用管理 ツール」連動）：

```
採用管理ツールで、採用業務を最大50%削減
中小企業の人事担当者向けのATS。スプレッドシート管理を卒業。

[3分で資料請求]   📞 [電話相談（営業時間 9-18時）]

★1,200社が導入   ★Pマーク取得
```

## Bad Pattern

- 広告とLP FVのメッセージマッチ崩壊（「採用管理」広告で「働き方改革」LP等）
- ロード遅い（広告品質スコア低下）
- 信頼バッジなし
- CTA1つのみ

## Design Notes

- LCP < 2.5s 必達（広告品質スコア対策）
- 主CTA + 副CTA（電話）
- SP最適化（広告流入の80%）
- 追従CTA配置

## Measurement Notes

- UTM パラメータ取得
- Google Ads CV計測（form_submit + Enhanced Conversion）
- Meta Pixel Lead イベント
- 電話クリック計測

## Performance Notes

- 画像最適化（WebP）
- フォント最適化
- クリティカルCSS インライン化
- defer / async でJS
