# FV Component: BtoB リード獲得LP

## Role in LP

3秒で「誰の・何の課題を・どう解決するか」を伝え、資料請求 / 無料相談 / デモ予約 に誘導する。

## Best Use Cases

- BtoB SaaS LP
- BtoBコンサルティング
- BtoB高単価サービス

## Required Elements

- メインキャッチ（ベネフィット型）
- サブコピー（補足機能）
- メインCTA（資料請求）
- 副CTA（デモ予約）
- 導入企業ロゴ列（社名 / ロゴ）or 実績数値
- ビジュアル（プロダクト画面 or 利用シーン）

## High Conversion Pattern

```
[業界 / 業務] の[課題]を、[改善幅 / 機能]で解決する[カテゴリ名]

[サブコピー: 補足説明]

[メインCTA: 無料で資料請求]   [副CTA: 無料デモを予約]

★[導入企業数] 社が導入   ★[認証バッジ]   ★[受賞]
```

実装例：

```
採用業務を、最大50%削減する採用管理ツール。
スプレッドシート管理を卒業し、応募者対応を自動化。

[無料で資料請求]   [3分で資料DL]

★ 1,200社が導入（自社調べ・2026年4月）   ★ Pマーク取得
```

## Bad Pattern

- 「最高の採用管理ツール」（抽象 / 根拠なし）
- 「○○Cloud」（商品名のみ）
- CTAなし
- 信頼バッジなし
- 課題訴求もベネフィット訴求もない

## Copy Examples

### ベネフィット型
- 「[業務] を [改善幅] する [カテゴリ]」

### 数字型
- 「[実績数] が選ぶ [カテゴリ] No.1」（要根拠）

### 課題訴求型
- 「[ターゲット] のための [カテゴリ]、ようやく登場」

## Design Notes

- 左: テキスト + CTA / 右: ビジュアル（PC）
- SP: 縦並び、CTAは画面内
- CTAコントラスト 4.5:1 以上
- 信頼バッジは下部または右下

## Measurement Notes

- メインCTA: `data-event="cta_click" data-position="fv"`
- 副CTA: `data-event="cta_click" data-position="fv-sub"`

## Checklist

- [ ] 3秒で意味が取れる
- [ ] ベネフィットまたは数字が含まれる
- [ ] CTA可視性（コントラスト・サイズ）
- [ ] 信頼バッジあり
- [ ] SP表示で破綻なし
- [ ] 計測 data 属性付与
