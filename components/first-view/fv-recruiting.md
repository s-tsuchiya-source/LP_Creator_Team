# FV Component: 採用LP

## Role in LP

求職者に「ここで働く未来」を3秒で伝え、応募 / カジュアル面談予約 へ誘導する。

## Best Use Cases

- エンジニア採用
- ハイクラス採用
- 新卒採用
- カジュアル面談募集

## Required Elements

- メインキャッチ（ミッション / 仕事の魅力）
- サブコピー
- メインCTA（応募 or 面談予約）
- 社員写真 or オフィス写真（許諾済）
- 信頼要素（受賞 / メディア掲載 / 数字）

## High Conversion Pattern

```
[ミッション or 仕事の本質]、[ターゲット]の挑戦の場

[サブコピー: 価値・特徴]

[メインCTA: カジュアル面談を予約する]

★社員数 / 受賞 / メディア掲載
```

実装例：

```
「誰かのために。」を、技術で形にする。
モダンな技術スタック × 自由な働き方 × 急成長中のSaaS。

[カジュアル面談を予約する]

★社員50名 / 創業5年 / Series B
```

## Bad Pattern

- 「成長できる環境です」（抽象）
- 「アットホームな職場」（誰でも言える）
- フリー素材の人物写真
- CTAが「応募」のみ（ハードル高）

## Copy Examples

### ミッション型
- 「[ミッション]に挑む、[ポジション]を募集」

### 仕事魅力型
- 「[仕事の魅力]が、ここにある」

### 強み型
- 「あなたの[強み]を、[業界 / 領域]で活かす」

## Design Notes

- 写真重要（実在社員 / オフィス）
- ロゴ + 主CTA + 副CTA（採用ピッチ動画 等）
- フォントは堅すぎず親しみやすく

## Measurement Notes

- 主CTA: `data-event="cta_click" data-position="fv-main"`
- 副CTA: `data-event="cta_click" data-position="fv-sub"`

## Legal Notes

- 男女・年齢制限表現NG
- 「絶対に活躍できる」等の保証NG
- 写真・氏名は同意取得済みのみ

## Checklist

- [ ] ミッション or 仕事魅力が伝わる
- [ ] 社員写真（許諾済）
- [ ] CTAが面談 等のローハードル選択肢を含む
- [ ] 法務（性別 / 年齢制限）チェック済
- [ ] 計測 data 属性付与
