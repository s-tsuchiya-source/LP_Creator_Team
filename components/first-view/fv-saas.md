# FV Component: SaaS LP

## Role in LP

ターゲットに「自分の業務がどう変わるか」を3秒で伝え、無料トライアル登録 / デモ予約に誘導する。

## Best Use Cases

- BtoB SaaS
- BtoC SaaS
- フリーミアム

## Required Elements

- メインキャッチ（ベネフィット型 + 数字）
- サブコピー（誰のためか）
- メインCTA（無料トライアル / 無料デモ）
- 副CTA（資料請求）
- プロダクト画面（モックアップ）
- 導入企業ロゴ列

## High Conversion Pattern

```
[業界]のチームを、[ベネフィット]でつなぐ

[サブコピー: 中小企業 / 大企業 / 特定業種 向け]

[14日間無料で試す（クレカ不要）]   [資料請求]

★[導入企業数] / [認証] / [受賞]

[プロダクト画面モックアップ]
```

実装例：

```
中小企業のチームを、シンプルにつなぐ。
プロジェクト管理 × タスク × チャットを1つに。

[14日間無料で試す（クレカ不要）]   [資料請求]

★800社が導入 / Pマーク取得 / Asia 50 受賞
```

## Bad Pattern

- 「最高のSaaS」（抽象）
- 機能羅列「APIあり / 連携可能 / 統計」（ベネフィット欠如）
- 価格を全く触れない（信頼欠如）
- プロダクト画面なし（イメージ湧かない）

## Copy Examples

### ベネフィット + 対象
- 「[業界]の[業務]を、[改善]するSaaS」

### Before / After
- 「[現状]だった業務を、[ToBe]へ」

### 速度訴求
- 「最短[期間]で[業務] を完了する[カテゴリ]」

## Design Notes

- プロダクト画面を大きく（信頼感）
- 「14日間無料」「クレカ不要」を強調
- CTAは「無料」を強調

## Measurement Notes

- 主CTA（無料トライアル）: `data-event="cta_click" data-position="fv-trial"`
- 副CTA（資料請求）: `data-event="cta_click" data-position="fv-doc"`
- プロダクト画面クリック計測（オプション）

## Checklist

- [ ] ベネフィットが具体的
- [ ] プロダクト画面あり
- [ ] 「無料」「クレカ不要」の明示
- [ ] 導入企業ロゴ列
- [ ] 計測 data 属性
