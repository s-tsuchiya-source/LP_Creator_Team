# Comparison Proof Patterns

## Role in LP

競合・代替手段との比較で、自社を選ぶ理由を証明する。

## Required Elements

- 比較表
- 客観的評価軸
- 自社が ○ になる項目だけでなく △ も含む（信頼性）

## High Conversion Pattern

### 比較対象
- 競合（A社 / B社等で匿名化、または許諾済名）
- 既存代替手段（人力 / Excel / 既存ツール）

### 評価軸（推奨）
- 主要機能
- 価格帯
- 導入期間
- サポート
- セキュリティ
- 中小企業対応 等

### 段階評価
- ◎ / ○ / △ / × の4段階
- 評価基準を明示

## High Conversion Layout

```
| 項目          | 自社  | A社  | B社  | 既存代替 |
|---------------|------|------|------|---------|
| 主要機能       | ◎    | ○    | △    | ×       |
| 価格帯         | 中小 | 大企業| 中  | 無料～   |
| 導入期間       | 2週間 | 1ヶ月 | 1ヶ月 | -       |
| データ移行     | 無料  | 有料  | 無料  | -       |
| サポート       | チャット | メール | メール | -      |

※ 各社の公式情報および公開資料（YYYY年MM月時点）
```

## Bad Pattern

- 自社が全項目 ○ / 競合が全項目 ×
- 競合社名を直接記載（許諾なし）
- 評価基準が主観的（「使いやすさ」だけ）
- 比較時期が古すぎ
- 競合の弱点を誇張

## Copy Examples

```markdown
## 他社との違い

[自社サービス] は、中小企業向けに特化した
価格設計と機能を提供しています。

| 項目 | 自社 | A社（大企業向け）| B社（無料系）|
|---|---|---|---|
| 月額（初年度） | ¥9,800〜 | ¥50,000〜 | 無料 |
| サポート | チャット即応 | メールのみ | コミュニティ |
| 中小企業対応 | ◎ | △ | ◎ |

※ 各社公式サイト（2026年4月時点）
```

## Design Notes

- 自社列を中央 or 左端でハイライト
- 段階評価は視覚的に分かりやすく
- 出典・時期を表下部に明記

## Measurement Notes

- 比較表閲覧計測（IntersectionObserver）

## Legal Notes

- 競合名は許諾 or 公開情報のみ
- 「No.1」「最高」「最安」は根拠付き
- 客観的事実の比較に限定（誹謗中傷NG）

## Checklist

- [ ] 客観的評価軸
- [ ] 自社劣位点も含む
- [ ] 出典・時期明記
- [ ] 競合名は安全
