# Proof Types

## Role in LP

信頼性を担保するための「証拠」の種類別パターン。

## Required Elements

LP内に最低3種類の証拠を配置することを推奨。

## High Conversion Pattern

### 1. 数値実績（出典付き）
- 「導入企業 1,200社（自社調べ・2026年4月時点）」
- 「採用工数 平均48% 削減（自社調査・100社平均・2025年12月）」

### 2. 第三者評価
- 「[賞名] 受賞（2025年・[主催団体]）」
- 「[認証機関] [認証名] 取得」
- 「[メディア名] に掲載（2025年X月号）」

### 3. お客様の声（許諾済）
- 顔写真 + 氏名 / イニシャル + 業種・役職
- 課題 → 効果のストーリー
- 数値効果（あれば）

### 4. 導入企業ロゴ列
- 6〜12社程度
- 業種バランス
- 許諾済みのみ

### 5. 比較表
- 客観的評価軸（機能 / 価格 / サポート / 期間）
- 段階評価（○ / △ / ×）
- 競合社名は許諾 or 公開情報のみ

### 6. 専門家のコメント
- 推薦者氏名・所属・推薦内容
- 利益相反の明示（あれば）

## Bad Pattern

- 「業界No.1」（根拠なし）
- 「最高品質」「最安値」（根拠なし）
- 出典なき数値
- 顧客の許諾なき名前掲載
- 競合への一方的批判

## Copy Examples

```
【数値実績の正しい書き方】

NG: 「業界No.1の採用管理ツール」

OK: 「[第三者調査会社] 調べ
    『中小企業向け採用管理ツール』顧客満足度No.1
    (2025年12月実績・対象者300名)」
```

## Design Notes

- 数値はバッジ風に大きく
- ロゴ列は均等配置
- お客様の声は顔写真と一緒
- メディア掲載はロゴと日付

## Measurement Notes

- 比較表の閲覧計測（IntersectionObserver）
- お客様の声カードのクリック計測

## Checklist

- [ ] 最低3種類の証拠配置
- [ ] 全数値に出典
- [ ] No.1表現に根拠
- [ ] 顧客許諾確認
- [ ] 比較項目が客観的
