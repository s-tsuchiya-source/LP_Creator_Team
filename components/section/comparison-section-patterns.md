# Comparison Section Patterns

## Role in LP

競合・代替手段との比較で、自社を選ぶ理由を視覚的に伝える。

## Required Elements

- 比較表
- 客観的評価軸
- 出典・時期明記

## High Conversion Pattern

### 競合との比較

```markdown
## 他社との違い

| 項目 | 自社 | A社 | B社 |
|---|---|---|---|
| 主要機能 | ◎ | ○ | △ |
| 中小企業対応 | ◎ | △ | ○ |
| 月額（初年度） | ¥9,800〜 | ¥50,000〜 | ¥29,800〜 |
| 導入期間 | 最短2週間 | 1〜2ヶ月 | 1ヶ月 |
| サポート | チャット即応 | メールのみ | メールのみ |

※ 各社公式情報および公開資料（2026年4月時点）
```

### 既存代替手段との比較

```markdown
## こんな方法、もう不要です

| 項目 | スプレッドシート | 人力管理 | [自社サービス] |
|---|---|---|---|
| 工数 | 多い | 多い | 少ない |
| 抜け漏れ | あり | あり | なし |
| 経営報告 | 手作業 | 手作業 | 自動 |
| 拡張性 | 低 | - | 高 |
```

## Bad Pattern

- 自社全項目 ○ / 競合全項目 ×（信頼欠如）
- 評価基準が主観的（「使いやすさ」のみ）
- 競合社名を許諾なく直接記載
- 古い情報（時期記載なし）
- 競合の弱点を誇張

## Copy Examples

### 比較表の補足コピー
```
※ 各社公式サイト（2026年4月時点）
※ プラン比較は各社「中小企業向け」プランで実施
※ 機能比較は当社調査による評価です
```

## Design Notes

- 自社列を中央 or 左でハイライト
- 段階評価は記号 + テキスト
- 出典を表下部に小さく
- アイコン併用で視認性UP

## Measurement Notes

- 比較表閲覧計測（IntersectionObserver）
- スクロール深度

## Legal Notes

- 競合社名は許諾 or 公開情報のみ
- 「No.1」「最高」「最安」は根拠付き
- 客観的事実の比較に限定
- 誤認リスクを排除

## Industry Notes

### BtoB SaaS
- 機能 + 価格 + サポート で比較
- 中小企業特化を強調

### コンサル
- 支援範囲 + 期間 + 料金で比較
- 業界特化を強調

### EC / 物販
- 価格 + 成分 + 配送で比較
- 法的に問題ない範囲

## Checklist

- [ ] 客観的評価軸
- [ ] 自社劣位点を含む
- [ ] 出典・時期明記
- [ ] 競合名が安全
- [ ] 法務リスクなし
