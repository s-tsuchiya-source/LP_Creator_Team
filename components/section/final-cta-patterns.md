# Final CTA Section Patterns

## Role in LP

LP最後の決断ポイント。ベネフィット再訴求 + 限定性 + 安心要素 + CTA で行動を促す。

## Required Elements

- 見出し（再訴求）
- ベネフィット箇条書き 3個
- 限定性 / 緊急性（実態に即す）
- メインCTA + サブCTA
- 安心要素（保証・解約条件）

## High Conversion Pattern

### BtoB SaaS

```markdown
## [サービス名] で、採用業務を変えませんか？

✓ 採用工数を最大50%削減
✓ 応募者対応の質を向上
✓ 経営層への報告が自動化

導入企業 1,200社が選ぶ採用管理ツール

[今すぐ無料で資料請求]   [無料デモを予約]

★ 14日間無料トライアル / クレジットカード不要
★ いつでも解約可能
★ 専任担当による導入支援
```

### 採用LP

```markdown
## あなたの挑戦を、待っています

✓ モダンな技術スタック × 自由な働き方
✓ 急成長中のSaaSで、裁量ある仕事
✓ 個人の成長を全力支援する制度

カジュアル面談から、気軽に始めませんか？

[カジュアル面談を予約する]   [メンバーと話す]

★ 業務内容・カルチャーを気軽にお聞きください
★ 応募ではないので、お断りも歓迎
```

### EC / 物販

```markdown
## いまなら[特典]付き

✓ [商品名] 30%OFF
✓ 送料無料
✓ 30日間返金保証

[今すぐ購入]   [カートに入れる]

★ 期間限定: 2026年X月X日まで
★ クレジット / 後払い対応
```

## Bad Pattern

- ベネフィット再訴求なし
- 限定性が嘘（常時セール）
- 安心要素なし
- CTA1つのみ
- 「もっと詳しく」のみ（決断促進弱い）

## Copy Examples

### 見出しパターン
- 「[サービス名] で、[業務 / 課題]を変えませんか？」
- 「あなたの[目標] を、いま始めませんか」
- 「最後のチャンスです」（実態に即す場合）

### 限定性パターン
- 「期間限定: 2026年X月X日まで」
- 「先着[数] 社限定」
- 「[特典] は[期間] のみ」

### 安心要素パターン
- 「クレジットカード不要」
- 「いつでも解約可能」
- 「30日間返金保証」
- 「専任担当による導入支援」

## Design Notes

- 背景色を変えて注目度UP
- CTA を大きく目立たせる
- 安心要素を CTA直下にバッジ風配置
- ベネフィットは✓アイコン

## Measurement Notes

- メインCTA: `data-event="cta_click" data-position="final-main"`
- サブCTA: `data-event="cta_click" data-position="final-sub"`

## Legal Notes

- 限定性は実態に即す（嘘NG）
- 「絶対」「必ず」の保証NG
- 返金保証は実行可能範囲

## Checklist

- [ ] ベネフィット再訴求
- [ ] 限定性（実態に即す）
- [ ] 安心要素3つ以上
- [ ] CTA2つ以上
- [ ] 計測属性
