# Flow Section Patterns

## Role in LP

行動後の流れを可視化し、「申し込んだら何が起きるのか」の不安を解消する。

## Required Elements

- 4〜5ステップ
- 各ステップに所要時間
- 担当者・役割明示

## High Conversion Pattern

### BtoB資料請求 → 商談

```markdown
## 導入までの流れ

### Step 1: 資料請求（30秒）
フォーム入力 → 即座にメールで資料受領

### Step 2: 個別相談（30〜60分）
オンラインで貴社課題をヒアリング

### Step 3: デモ・トライアル開始（1〜2週間）
無料で実環境で試用

### Step 4: 契約・本番導入（1ヶ月）
契約 → 設定 → スタッフ研修

### Step 5: 運用開始
専任担当が継続サポート
```

### 採用LP カジュアル面談

```markdown
## 選考の流れ

### Step 1: カジュアル面談（30分）
気軽な雑談形式。質問・相談歓迎

### Step 2: エントリー（任意）
興味があれば応募。お断りも歓迎

### Step 3: 1次面接（60分）
業務内容・期待役割の擦り合わせ

### Step 4: 最終面接（60分）
代表 or 部門責任者との対話

### Step 5: オファー
1週間以内に結果連絡
```

## Bad Pattern

- 抽象的「お申し込み → 利用開始」のみ
- 所要時間記載なし
- 担当者不明
- 不安箇所を見えにくくする

## Design Notes

- ステップを横並び（PC）/ 縦並び（SP）
- 各ステップにアイコン
- 矢印で順序明示
- 所要時間を明示

## Measurement Notes

- セクション閲覧計測

## Industry Notes

### BtoB
- 営業プロセス全体（リード → 商談 → 受注）
- 各ステップの担当者明示

### 採用
- カジュアル面談を最初に配置（ハードル低）
- 結果連絡期間明示

### EC / 物販
- 注文 → 配送 → 受取 → サポート
- 配送日数明示

### スクール / 教育
- 体験 → 申込 → 受講開始 → 卒業 / 資格取得
- 受講期間 / サポート期間明示
