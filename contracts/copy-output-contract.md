# Copy Output Contract

## Purpose

LPコピーの出力形式を固定化する。

## Required Sections

- ファーストビュー（メインキャッチ / サブコピー / CTA文言）
- 問題提起 / 共感 / 解決策
- 強み・特徴
- 実績・お客様の声
- 比較表
- FAQ（5〜10件）
- 最終CTAセクション
- マイクロコピー（フォーム / エラー / サンクス）

## Required Fields

- メインキャッチ
- サブコピー
- 全セクション本文
- FAQ最低5件
- CTAボタン文言（最低2種）
- フォームエラーメッセージ

## Optional Fields

- ストーリー型のセクション
- 動画スクリプト

## Format Rules

- `templates/lp-copy-template.md` に準拠
- 全コピー本文を記載（要約禁止）
- 数値・実績は出典付き
- No.1表現は根拠付き

## Acceptance Criteria

- 「誰向け / 何が得られる / なぜ信頼 / なぜ今」が明確
- ベネフィットと証拠がセット
- CTAが具体的（「お問い合わせ」だけでない）
- 誇大表現なし
- トーンが一貫

## Rejection Criteria

- 抽象的訴求（「品質が良い」だけ）
- ベネフィット欠如
- 出典なき数値
- 法令違反表現
