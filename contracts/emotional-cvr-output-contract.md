# Emotional CVR Output Contract

## Purpose

感情設計とCV導線を紐づけるための出力 contract。
LPの 9段階感情ジャーニーが、各セクション・コピー・CTAに反映されているかを保証する。

## Required Sections

### 1. 流入時の心理
- 「これ何のサービス？」「自分に関係ある？」
- 対応LP要素: FV / 信頼バッジ / ターゲット明示

### 2. FVで与える第一印象
- 「3秒で価値が分かる」
- 対応コピー: メインキャッチ / サブコピー
- 対応CTA: メインCTA文言

### 3. 課題共感で発生させる感情
- 「分かってくれている」「これは深刻な問題」
- 対応LP要素: 課題提起セクション
- 言語化されていない悩みの3〜5個

### 4. 解決策提示で発生させる期待
- 「これなら解決できそう」
- 対応LP要素: 解決策セクション / 機能 × ベネフィット
- 数値根拠（出典付き）

### 5. 証拠提示で発生させる安心
- 「実在の会社が、実在の成果」
- 対応LP要素: 実績 / 事例 / 認証 / メディア掲載

### 6. 比較・事例で発生させる納得
- 「これを選ぶ理由が分かる」
- 対応LP要素: 比較表 / 同業種事例

### 7. FAQで解消する不安
- 価格 / 導入 / 効果 / 運用 / 社内 / 失敗 の6カテゴリ
- 対応LP要素: FAQ 5〜10件

### 8. CTA直前に作る行動意欲
- 「今動く理由」「ハードル低い行動」
- 対応LP要素: 最終CTAセクション / オファー再訴求 / 限定性

### 9. CV後に期待させる体験
- 「申込後の流れ」「営業されすぎない安心」
- 対応LP要素: サンクスページ / 流れ図 / FAQ

### 10. 感情とCV導線の接続
- 9段階のジャーニーが断絶していないか
- 各段階の感情が次の段階に繋がっているか

## Required Fields

- 各段階の心理 + 対応LP要素の明示
- 言語化されていない悩み 3〜5個
- 数値根拠の配置箇所
- FAQ 6カテゴリ網羅
- CTA直前の行動意欲ブースト
- サンクスページ設計

## Optional Fields

- 業種別感情パターン
- 各段階の閲覧時間目安

## Format Rules

- 9段階を明確に分ける
- 各段階で「心理」「LP要素」「コピー例」を提示
- ジャーニーが断絶しないよう、前後の段階との接続を意識

## Acceptance Criteria

- 9段階全てが設計されている
- 各段階に対応LP要素・コピーがある
- ジャーニーが断絶していない
- 期待 / 信頼 / 不安解消のバランス
- CV直前の心理ハードル下げ
- CV後の安心感

## Rejection Criteria

- 段階欠落
- 期待だけが先行（不安解消なし）
- 信頼要素ばかりでベネフィット欠如
- 機能羅列のみ
- CTA直前のハードル下げなし
- CV後の流れ不明
