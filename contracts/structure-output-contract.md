# Structure Output Contract

## Purpose

LPセクション構成書の出力形式を固定化する。

## Required Sections

- セクション順序（10〜13個）
- 各セクションの目的・必須要素
- CTA配置
- 中盤離脱対策
- 視線誘導方針

## Required Fields

- 全セクション名
- 各セクションの目的
- 各セクションの必須要素
- CTA配置箇所（最低3箇所）
- 視線誘導タイプ（Z型 / F型 / 中央集中）

## Optional Fields

- 装飾要素
- アニメーション方針

## Format Rules

- 表形式または番号付きリスト
- 各セクションに目的・必須要素・CTA有無を明示
- PC / SP の差分があれば明示

## Acceptance Criteria

- FVに価値・CTA・信頼バッジが含まれる
- 問題→解決→証拠→FAQ→CTA の流れがある
- 中盤CTA最低1箇所
- セクション数10〜13

## Rejection Criteria

- セクション数極端（5以下 / 15以上）
- CTA配置1箇所のみ
- 証拠なしで強み羅列
- FAQ なし
