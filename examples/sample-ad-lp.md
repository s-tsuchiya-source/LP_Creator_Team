# サンプル: 広告流入LP（架空サービス）

> このサンプルは架空サービス「LegalCheckPro（法務文書AIチェック）」の Google 広告流入LPです。

## 依頼例

```
LP_Creator_Team の lp-director を起点に、広告流入LPを作ってください。

商材: LegalCheckPro（法務文書AIチェックSaaS / 架空）
広告キーワード: 「契約書 チェック」「リーガルチェック AI」
ターゲット: 中小企業の法務担当者・経営者
目的: 資料請求 + デモ予約

速度重視（LCP < 2.5s）、計測重視（GA4 / Google Ads CV）でお願いします。
```

## 想定出力（要約）

### Strategy
- メッセージマッチ: 広告「契約書 チェック」→ FV「契約書チェックを、AIで」
- 主訴求: 「弁護士チェック前の一次チェックを、AIで瞬時に」
- 差別化: 「中小企業向けの価格」「日本法準拠」

### Structure
（広告流入特化）
1. FV: 強烈なベネフィット + 主CTA + 補助CTA（電話）
2. 課題（広告キーワードと連動）
3. 解決策（AI機能のベネフィット）
4. 導入実績
5. 比較表（既存代替: 弁護士のみチェック / 自社人力）
6. 価格
7. FAQ
8. 最終CTA

### Copy（抜粋）
- メインキャッチ: 「契約書のチェック工数を、最大70%削減（自社調べ）」
- サブコピー: 「AIが見落としがちな条項を瞬時に検出。最終チェックは弁護士へ。」
- CTA: 「無料で資料請求」「無料デモを予約」

### Performance
- LCP < 2.5s（画像WebP化 / フォント最適化 / クリティカルCSS）
- CLS < 0.1
- INP < 200ms
- LP合計サイズ目安: 800KB

### Measurement
- GA4 イベント: cta_click / form_submit
- Google Ads CV: form_submit + Enhanced Conversion
- Meta Pixel（並行運用なら）: Lead

### 法務
- 「法務」ドメインだが医療・金融ではないため特別法令はなし
- ただし「絶対に法的リスクなし」等の保証表現NG
- 「最終チェックは弁護士へ」と注釈必須（誤認回避）

## 注意

- 広告キーワード変更時はFV再調整
- 数値は出典明記
- 弁護士業務との誤認を避ける表現

## このサンプルの活用方法

広告コピーとLP FV のメッセージマッチを意識する。
