# サンプル: BtoB SaaS LP（架空サービス）

> このサンプルは架空サービス「TalentFlow（採用管理ツール）」を題材にした LP 制作例です。実在のサービスや実績を含みません。

## 依頼例

```
LP_Creator_Team の lp-director を起点にしてください。

商材: TalentFlow（採用管理SaaS／架空）
目的: 資料請求獲得
ターゲット: 中小企業の人事担当者
法務リスク: なし

戦略 → 構成 → コピー → デザイン方針 → 計測 までお願いします。
実装は後段で別途依頼します。
```

## 想定出力（要約）

### Strategy
- ペルソナ: 30〜45歳の人事担当者、採用業務に月100時間以上、ATS未導入
- 主訴求: 「採用業務を一元管理し、工数を削減」
- 副訴求: 「直感的UIで現場が定着する」
- 差別化: 「中小企業特化の価格帯 + API連携」
- オファー: 14日間無料トライアル / 資料請求

### Structure
1. ファーストビュー（ベネフィット型キャッチ + CTA + 導入企業ロゴ）
2. 課題訴求（5つの悩み）
3. 解決策（3つのアプローチ）
4. 主要機能 × ベネフィット（5機能）
5. 導入事例（3社）
6. 競合比較表
7. セキュリティ
8. 価格プラン（3プラン）
9. FAQ（7件）
10. 最終CTA

### Copy（抜粋）
- メインキャッチ: 「採用業務を、最大50%削減。中小企業のためのATS『TalentFlow』」
- サブコピー: 「散らばった応募者管理を一元化。人事の本来の仕事に集中できる環境を。」
- CTA: 「無料で資料を受け取る」

### Design
- トーン: プロフェッショナル / 親しみやすい
- メインカラー: #2563EB（信頼感）
- アクセントカラー: #F59E0B（CTA）
- フォント: Noto Sans JP / Inter

### Measurement
- GA4 イベント: page_view / cta_click / form_submit
- Google Ads CV: form_submit
- Enhanced Conversion: 有効

### 注意点
- 「業界No.1」「最高」表現は使用していない
- 「導入企業数」は架空数値であり、実装時は実数値・出典に置き換える必要あり
- 顧客実名・写真は許諾済みのものに置き換える

## このサンプルの活用方法

実案件で同様の依頼をする際の参考にしてください。実数値・実顧客名は **必ず** 実案件のデータに置き換えること。
