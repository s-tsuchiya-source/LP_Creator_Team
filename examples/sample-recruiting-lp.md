# サンプル: 採用LP（架空企業）

> このサンプルは架空企業「BlueOcean Tech（架空のスタートアップ）」のエンジニア採用LP制作例です。

## 依頼例

```
LP_Creator_Team の lp-director を起点にしてください。

会社: BlueOcean Tech（架空 / SaaS / 50名）
募集: バックエンドエンジニア（3〜5年経験）
CV: カジュアル面談予約
価値: モダンな技術スタック / リモートワーク / 成長機会

法務観点（性別 / 年齢の制限表現）に注意してください。
```

## 想定出力（要約）

### Persona（求職者ペルソナ）
- 30〜35歳、Webエンジニア3〜5年経験
- 現職: SI / 受託開発に飽き、自社サービスへ
- 動機: 技術成長・裁量・働き方
- 不安: 入ってからのギャップ・カルチャー

### Strategy
- 主訴求: 「自社プロダクト × モダン技術 × リモート」
- 副訴求: 「少人数だから挑戦できる」
- CTA: カジュアル面談（応募よりハードル低）

### Structure
1. ファーストビュー（ミッション + 写真 + CTA）
2. 「こんな方を求めています」
3. 仕事内容・期待役割
4. 技術スタック
5. 1日の流れ
6. 社員インタビュー（3名）
7. 制度・福利厚生
8. 選考フロー
9. FAQ
10. 最終CTA（カジュアル面談）

### Copy（抜粋）
- メインキャッチ: 「『誰かのために。』を、技術で形にする」
- CTA: 「カジュアル面談を予約する」

### 法務チェック
- 性別・年齢制限表現なし
- 「絶対に成長できる」等の保証表現なし
- 写真・氏名は社内同意取得前提

## 注意

- 写真・社員氏名・実年齢は同意取得が前提
- 給与レンジは法令・競合相場を確認して掲載

## このサンプルの活用方法

実案件は実在社員の同意・実給与レンジに置き換えてください。
