# experience-design/ — 体験価値・感情・CV接続レイヤー

## 目的

LPの目的は単なる情報伝達ではなく、見たユーザーが
- 自分に関係があると感じる
- メリットを直感的に理解する
- 導入後の良い未来を想像できる
- 不安より期待が上回る
- 自然にCVへ進みたくなる

という心理状態を作ることです。

このディレクトリは、その**心理変化を設計するための中核ナレッジ**を集約します。

## LPにおける体験価値とは

LPの品質は次の式で決まります：

```
LPの品質 = 情報品質 × 信頼品質 × 体験価値 × 期待感 × CV導線
```

体験価値とは、ユーザーが LP を読み進める中で得る「期待」「安心」「納得」「自分ごと化」「行動意欲」の総量です。

## CV最大化と感情設計の関係

CV は論理だけでは起きません。

ユーザーは
1. **論理**でメリットを理解し
2. **感情**で期待・安心・納得を得て
3. **行動**に踏み出します

LP_Creator_Team では、戦略・構成・コピー・デザイン・実装・計測に加えて、**感情設計レイヤー**を必ず通過させます。

## BtoB でも「わくわく感」が重要な理由

BtoBは合理判断と思われがちですが、実際は

- 「成果が出そう」（期待）
- 「失敗しなさそう」（安心）
- 「社内で説明できそう」（納得）
- 「上司に推せる」（自信）

という**感情** で意思決定が動きます。

ただし、わくわく感は**派手な装飾や煽りではなく**、具体的なメリット・信頼できる証拠・導入後の未来・安心できるプロセスで作ります。
