# 不安から期待への転換設計

ユーザーの「不安」を、LPで「期待」に変換します。
不安が残ったままだとCVは起きません。

## 不安の6カテゴリ

### 1. 価格不安
- 「高そう」
- 「予算オーバー」
- 「上司に通らない」
- 「費用対効果が見えない」

### 2. 導入不安
- 「導入に時間がかかりそう」
- 「現場が定着しない」
- 「データ移行が大変」
- 「システム連携が複雑そう」

### 3. 効果不安
- 「本当に効果あるのか」
- 「自社に合うのか」
- 「同業種で実績ある？」
- 「期待外れだったら」

### 4. 運用負荷不安
- 「使いこなせない」
- 「専任担当が必要なのでは」
- 「現場の教育が大変」
- 「アップデート対応が負担」

### 5. 社内説明不安
- 「上司に説明できない」
- 「経営層の決裁が通らない」
- 「他部門の理解が得られない」
- 「比較資料がない」

### 6. 失敗不安
- 「失敗したら責任問題」
- 「導入後に後悔したくない」
- 「契約してから後戻りできない」
- 「他社に乗り換えづらい」

## 各不安を期待に転換する方法

### 1. 価格不安 → ROI / 分割の期待

**転換コピー例**
```
月額 9,800円は、月50時間の業務削減に相当します。
時給3,000円換算で、月15万円の効果。

つまり、月額の約15倍のROIが期待できます。

※ 効果は規模により変動します
```

**転換要素**
- ROI試算
- 分割払い
- 14日間無料トライアル
- 解約条件の柔軟性

### 2. 導入不安 → 安心の期待

**転換コピー例**
```
最短2週間で導入完了。

✓ 専任担当が初期設定を支援（無料）
✓ データ移行は自動 / または代行サービス（無料）
✓ 既存ツールとのAPI連携対応
✓ 導入企業の85%が「想定より簡単」と回答（自社調べ）
```

**転換要素**
- 導入フロー図
- 専任担当紹介
- 同業種の導入事例

### 3. 効果不安 → 確信の期待

**転換コピー例**
```
あなたの業界（[業種名]）で、すでにこんな成果が出ています：

- A社（従業員50名）: 採用工数 60%削減
- B社（従業員200名）: 応募数 1.5倍
- C社（従業員500名）: 内定承諾率 30%向上

※ 自社調べ・2025年実績。効果には個人差があります
```

**転換要素**
- 同業種・同規模事例
- 数値実績（出典付き）
- 効果保証 / 返金保証（範囲内）

### 4. 運用負荷不安 → サポートの期待

**転換コピー例**
```
「使いこなせるか不安」という声を、よくいただきます。

そのため、[サービス名] では：

✓ 操作研修（オンライン・無料・何回でも）
✓ チャットサポート（平日9-18時・即応）
✓ FAQ・動画チュートリアル（24時間）
✓ 導入後3ヶ月の伴走支援

「現場が定着するまで支援します」が私たちの約束です。
```

**転換要素**
- サポート体制
- 教育コンテンツ
- 伴走支援
- 現場定着率データ

### 5. 社内説明不安 → 推進力の期待

**転換コピー例**
```
社内検討用の資料を、無料でご用意しています：

✓ 機能比較表（競合A・B社との比較）
✓ ROI試算シート（あなたの会社規模で計算）
✓ 同業種の導入事例集
✓ セキュリティチェックリスト（情シス向け）
✓ 経営層向け1pサマリー

社内会議で、そのままお使いいただけます。
```

**転換要素**
- 社内検討資料
- ROI試算
- 比較表
- セキュリティ資料

### 6. 失敗不安 → 安全の期待

**転換コピー例**
```
万が一、合わなかった場合：

✓ 月単位の解約OK（最低利用期間なし）
✓ データエクスポート機能（CSV / API）
✓ 14日間無料トライアル中の解約自由

「契約後も、自由に判断してください」という設計です。
```

**転換要素**
- 解約条件の明示
- データポータビリティ
- 無料トライアル
- 返金保証（範囲内）

## 転換の配置

LP内で不安解消を配置する位置：

| セクション | 解消する不安 |
|---|---|
| 強み | 効果不安 |
| 実績・事例 | 効果不安 / 失敗不安 |
| 比較表 | 社内説明不安 |
| 価格 | 価格不安 |
| 導入フロー | 導入不安 |
| サポート体制 | 運用負荷不安 |
| FAQ | 全不安カテゴリ |
| 最終CTA直前 | 失敗不安 |

## FAQ で各不安を解消する

FAQは「不安解消」専用セクション。各カテゴリから1つずつ取り上げる。

```markdown
**Q. 月額9,800円は、中小企業にとって高くないですか？**
A. ROI試算では、月50時間の工数削減で約15万円相当の効果。
   月額の約15倍のリターンが期待できます。
   詳細は、無料の試算資料でご確認ください。

**Q. 導入時間はどれくらいかかりますか？**
A. 最短2週間です。専任担当が支援します。
   データ移行が必要な場合も、無料で代行可能です。

**Q. 効果は本当に出ますか？**
A. 同業種の導入事例（A社、B社、C社）で、
   平均60%の工数削減実績があります（自社調べ）。
   ただし効果は個人差・業種差があります。

**Q. 解約は自由にできますか？**
A. 月単位で解約可能です。最低利用期間はありません。
   データはいつでもCSVエクスポート可能です。
```
