# hearing-sheet.md — ヒアリングシート

LP制作依頼時、`lp-director` が最初に確認する項目。**最大12問** で必須情報を取得し、不足は仮置き補完する。

## 必須12項目

| # | 質問 | 必須度 | 補完可否 |
|---|------|---|---|
| 1 | LPで扱う商材・サービス名は何ですか？ | ★★★ | × |
| 2 | LPの目的は何ですか？（資料請求 / 問い合わせ / 申込 / 採用応募 等） | ★★★ | × |
| 3 | 最終CV地点は何ですか？（フォーム送信 / 電話 / 来店予約 等） | ★★★ | × |
| 4 | 主なターゲットは誰ですか？（業種 / 役職 / 年齢層 等） | ★★★ | △ |
| 5 | ターゲットが抱えている悩みは何ですか？ | ★★ | △ |
| 6 | サービスの主な提供価値は何ですか？ | ★★★ | △ |
| 7 | 競合・代替手段は何ですか？ | ★★ | △ |
| 8 | 強み・差別化要素は何ですか？ | ★★ | △ |
| 9 | 実績・事例・数字・証拠はありますか？ | ★★ | △ |
| 10 | 希望するトーンやデザインイメージはありますか？ | ★ | ○ |
| 11 | 参考LPや競合LPはありますか？ | ★ | ○ |
| 12 | 希望する出力形式は何ですか？（戦略のみ / コピー込み / 実装込み 等） | ★★ | × |

凡例:
- 必須度: ★★★=必須 / ★★=可能な限り取得 / ★=任意
- 補完可否: ×=必ず確認 / △=仮置き可（要明示） / ○=仮置き可

## 法務リスク確認（追加質問）

以下に該当する場合は12問とは別に確認：

- 医薬品・医療機器・健康食品 → 薬機法
- 美容・脱毛・痩身 → 薬機法・景表法
- 金融・投資 → 金商法
- 不動産 → 宅建業法
- 教育・資格 → 特商法・割販法
- BtoB高単価 → 特商法

## ヒアリング後の仮置き補完ルール

### 補完してよい項目
- ペルソナの細部（年収・既婚/未婚 等）
- 競合の具体名（業界一般論で代替）
- デザイン詳細（業界標準で仮置き）
- 数字の幅（範囲表現で代替: 「数百社規模」等）

### 補完してはいけない項目
- CV地点
- ターゲットの大枠
- 商材の機能・効果
- 実績・数字の断定値
- 法的リスクのある表現

## ヒアリング会話の進め方

1. ユーザーの初回依頼を読み、既に判明している項目をマーク
2. **不足12項目以内** を1メッセージで質問
3. ユーザー回答を整理し `outputs/01_brief/` に保存
4. 不足分のうち補完可能なものは仮置きで前進
5. 補完不可分のみ追加質問

## 出力フォーマット例

```markdown
## 確認済み項目
- 商材: 〇〇クラウド
- 目的: 資料請求獲得
- CV: フォーム送信
- ターゲット: 中小企業の人事担当者
...

## 仮置き項目
- ペルソナ年齢: 30〜45歳（業界標準）
- 競合: 一般的な人事SaaS各社
...

## 確認が必要な項目
- 実績数字の根拠
- 薬機法該当の可能性
```
