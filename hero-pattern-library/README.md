# Hero Pattern Library

LPのヒーロー（FV）パターン10種を網羅したライブラリ。
`lp-hero-art-director` がヒーロー選定時に最低3パターンを比較する基準。

## 10パターン

| # | パターン | 概要 |
|---|---|---|
| 1 | product-ui-preview-hero | ダッシュボード/プロダクト画面 |
| 2 | photo-annotation-hero | 業務シーン写真 + 注釈 + KPI |
| 3 | editorial-typography-hero | タイポ主体（雑誌的） |
| 4 | before-after-hero | Before/After 比較 |
| 5 | workflow-diagram-hero | 業務フロー図解 |
| 6 | customer-scene-hero | 顧客シーン（採用・教育向き） |
| 7 | report-ledger-hero | 帳票・レポート見せる |
| 8 | video-demo-hero | 動画デモ（自動再生・サイレント） |
| 9 | problem-visualization-hero | 課題を視覚化（インフォグラフィック） |
| 10 | no-image-bold-copy-hero | 画像なし、大胆コピー主体 |

## 利用ルール

- LP制作時に最低3パターン比較
- 5軸採点（文脈 / 共感 / CV / 差別化 / 実現性）
- 選定理由を `outputs/05_design/hero-pattern-selection.md` に記録
- ダッシュボード型をデフォルト化しない
