# Editorial Typography Hero

## 概要

タイポグラフィを主役にした、雑誌的・哲学的なヒーロー。画像最小限。

## Best Use Cases

- 哲学・ミッション系商材
- 採用LP（カルチャー訴求）
- 高単価コンサル
- ブランド訴求

## NG Use Cases

- 商材の機能訴求が必要
- 数字訴求が CV に直結する商材
- ペルソナが画像で意思決定する商材

## 構成要素

```
- 大胆なタイポ（48〜96px）
- ブランド単色 or 微妙なテクスチャ背景
- 短いメインキャッチ（5〜15文字）
- 補足コピー（30〜50文字）
- メインCTA
- 信頼バッジ最小限
```

## CV貢献

- ブランド表現
- 哲学・ミッション伝達
- 差別化（SaaS 一般感を超える）

## 例

```html
<section class="hero hero-editorial">
  <h1 class="hero-title-big">
    採用は、<br>
    技術で変えられる。
  </h1>
  <p class="hero-sub">
    導入1,200社が選ぶ採用管理ツール
  </p>
  <a href="#form" class="btn btn-primary">資料を受け取る</a>
</section>
```

## 採用判断

```
文脈: 業種依存
共感: 4（哲学に共感）
CV: 3〜4（機能訴求弱い）
差別化: 5
実現: 5
合計: 17〜19
```
