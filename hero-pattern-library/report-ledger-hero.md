# Report Ledger Hero

## 概要

帳票・レポート・分析結果をヒーローに配置するパターン。

## Best Use Cases

- データ分析 SaaS
- レポート自動生成系
- 経営層向け商材（数字で意思決定）
- 帳票・会計系商材

## NG Use Cases

- 帳票が訴求装置にならない商材
- 数字訴求が弱い商材

## 構成要素

```
- 帳票・レポートのモックアップ
  - 主要数字を強調
  - グラフ・チャート
- 注釈で要素を強調
- メインキャッチ
- メインCTA
```

## CV貢献

- 「こんな分析ができる」即時理解
- 数値の説得力
- データ駆動の意思決定訴求

## 例

```
[ダッシュボード レポート モックアップ]
  - 月次採用工数推移グラフ
  - 媒体別CVRサマリ
  - 採用単価トレンド

注釈:
  - 「自動生成」（リアルタイム）
  - 「経営層への報告がワンクリック」
```

## 採用判断

