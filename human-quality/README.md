# Human Quality Layer

Human Quality Layer は、AIが作ったように見えるLPを避け、人間が現場を理解して作ったような自然さ・具体性・温度感・信頼感を担保するための品質レイヤーです。

## Purpose

- AIっぽい文章を検出する
- 汎用的な構成を避ける
- 具体性・現場感・読み手の本音を加える
- 画像・イラスト・デザインのテンプレート臭を排除する
- CVへ進む自然な理由を作る

## Core Principle

最高品質のLPは、整っているだけでは不十分です。読み手が「これは自分の状況を分かっている」と感じ、メリット・期待・安心を持って行動できる必要があります。

## Required Checks

- 抽象語が多すぎないか
- どの会社にも当てはまるコピーではないか
- 読み手の具体的状況が見えるか
- 画像・図解が理解を助けているか
- CTAが自然な次アクションになっているか
