# AI Smell Detection List

AIっぽさを検出するためのチェックリスト集。

## レベル1: 単語レベル検出

### 危険語（雑に使わない）

```
❌ 最適化
❌ 革新的
❌ 寄り添う
❌ 課題解決
❌ シームレス
❌ ホリスティック
❌ 業界をリード
❌ お客様第一
❌ 唯一無二
❌ 業界初
❌ 最高品質
❌ 革命的
❌ パラダイムシフト
```

これらが連続して3つ以上使われていれば AI 臭い。

### 単独でも危険な表現

```
❌ 「業界No.1」（根拠なし）
❌ 「絶対」「100%」「必ず」
❌ 「最高」「最強」「最安」（根拠なし）
❌ 「圧倒的」「劇的」（根拠なし）
```

### 使ってよい場合の条件

これらの表現を使う場合は、以下を必ず添える：

- 第三者調査の出典
- 調査対象・調査範囲・時期
- 「※個人差・業種差があります」

## レベル2: 文レベル検出

### NG文構造

#### パターン1: 「〜を実現します」連発
```
DXを実現します。
業務効率化を実現します。
成長を実現します。
最適なソリューションを実現します。
```

→ 何をするか不明。

#### パターン2: 「お客様」「ユーザー」抽象化
```
お客様のニーズに応えます。
ユーザーの体験を最大化します。
すべての方に最適です。
```

→ 誰向けか不明。

#### パターン3: 機能だけ並べる
```
APIを提供します。
ダッシュボードがあります。
データ分析が可能です。
```

→ 「だから何？」となる。

#### パターン4: 文がきれいすぎる
```
私たちは、お客様の課題に寄り添い、
最適なソリューションを提供することで、
お客様の事業成長を実現します。
```

→ 文法は完璧だが、内容ゼロ。

## レベル3: セクションレベル検出

### NGセクション

#### NG-FV
- ターゲット明示なし
- 抽象的なメインキャッチ（「世界を変える」等）
- 信頼バッジなし
- AIっぽい青グラデーション hero

#### NG-課題提起
- 業界一般論（「業務効率を上げたい」だけ）
- 担当者の本音なし
- 数字なし

#### NG-解決策
- 機能羅列のみ
- 「だから何？」が伝わらない
- 数値根拠なし

#### NG-実績
- 「業界No.1」（根拠なし）
- 出典なき数値
- テンプレお客様の声（「すごく便利です！」だけ）

#### NG-比較表
- 自社全項目 ○ / 競合全項目 ×
- 評価軸が主観的
- 競合許諾なし社名記載

#### NG-FAQ
- 「ご安心ください」だけ
- 主要不安カテゴリ欠落
- 不利情報の隠蔽

#### NG-CTA
- 「お問い合わせ」だけ
- 「詳細はこちら」だけ
- 安心要素なし
- CV後の流れなし

## レベル4: 全体レベル検出

### 全体を通して

#### NG-1: 競合置き換えテスト失敗
LP全体を競合社名に置き換えても成立する。

#### NG-2: 業務シーンが描けていない
特定の業務瞬間が見えない。

#### NG-3: 担当者の本音がない
ペルソナの感情・心理が見えない。

#### NG-4: 数字の出典がない
全体的に数値根拠が薄い。

#### NG-5: 装飾的な画像ばかり
意味のあるビジュアルが少ない。

## 検出時の対応

### コピー修正フロー
1. 危険語をリストアップ
2. 該当箇所を `skills/09_humanization/anti-ai-copy-editing-skill.md` で修正
3. 競合置き換えテスト合格まで繰り返し

### デザイン修正フロー
1. AIっぽい画像をリストアップ
2. `skills/09_humanization/image-direction-humanization-skill.md` で再指示
3. 商材文脈に合うビジュアルに置換

## チェック実行例

### Before（AIっぽい）

```
私たちは、お客様の業務効率を最適化する革新的なソリューションを提供します。
業界をリードする最高品質のサービスで、お客様に寄り添い、課題解決を実現します。
お客様第一の姿勢で、シームレスな体験をお届けします。
```

### After（人間らしい）

```
中小企業の人事担当者向けに、採用管理ツール [サービス名] を提供しています。

「月曜の朝、応募者リストを見るのが憂鬱」
「上司から『あの応募者、返信した？』と聞かれてヒヤヒヤ」

そんな声から生まれたサービスです。

導入企業1,200社（自社調べ・2026年4月時点）の平均で、
月150時間 → 月60時間（60%削減）の効果を実現しています。

※ 効果には個人差・業種差があります
```

## チェックリスト
