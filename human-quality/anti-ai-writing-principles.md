# Anti-AI Writing Principles

AIっぽい文章を排除し、人間が現場を理解して書いたような文章にするための原則。

## AIっぽい文章の特徴

### 1. 抽象語が多い
- 「最適化」「革新的」「課題解決」「寄り添う」
- 「業界をリード」「最高のサービス」
- 「シームレス」「ホリスティック」

### 2. どの会社にも当てはまる
- 「お客様第一の姿勢」
- 「業界のリーディングカンパニー」
- 「お客様のニーズに応える」

### 3. 実体験がない
- 現場感ゼロ
- 業務の特定瞬間が描けない
- 担当者の本音がない

### 4. 主語が曖昧
- 「私たち」が誰を指すか不明
- 「ユーザー」「お客様」を一般化しすぎる
- 「あなた」と「みなさん」が混在

### 5. 一文がきれいすぎる
- 文法的に完璧
- 余白がない
- 余韻がない
- すべての文が同じテンポ

### 6. 「〜を実現します」が多い
- 「DXを実現します」
- 「業務効率化を実現します」
- 「成長を実現します」
→ 何をするかが見えない

### 7. 「最適」「革新」「伴走」「寄り添う」を雑に使う
- 「最適なソリューション」
- 「革新的なアプローチ」
- 「お客様に寄り添う」
→ 言葉が空回り

### 8. メリットではなく機能説明に終始
- 「APIを提供します」
- 「ダッシュボード機能があります」
- 「データ分析が可能です」
→ 「だから何？」となる

## 人間らしい文章の特徴

### 1. 読み手の具体的な状況が見える
- 「月曜の朝、スプレッドシートを開くのが憂鬱」
- 「『あの応募者、返信した？』と上司に聞かれるたびにヒヤヒヤ」
- 「夜眠れない時がある」

### 2. 本音がある
- 「失敗したくない」
- 「上司に怒られたくない」
- 「胸を張って報告したい」
- 「業務疲弊から解放されたい」

### 3. 言い切りと余白のバランス
- 完璧な文だけでなく、余白を残す
- 「〜かもしれません」「〜の場合もあります」
- 体言止めも使う

### 4. 不安や迷いを自然に扱う
- 「ご検討中の方も多いかと思います」
- 「いきなり契約は不安ですよね」
- 「営業されすぎないか心配な方へ」

### 5. 数字・事例・場面がある
- 抽象論ではなく具体例
- 「100社平均で60%削減（自社調べ）」
- 「A社（製造業100名）の場合」

### 6. 売り込みすぎない
- 「資料を見るだけでもOK」
- 「お断りも歓迎」
- 「営業電話はかけません」

### 7. CV後の安心がある
- 「申込後の流れ」明示
- 「営業電話なし」明示
- 「メルマガ任意」明示

## 抽象語 → 具体語 変換例

| ❌ AIっぽい抽象語 | ✅ 人間らしい具体語 |
|---|---|
| 業務効率化 | 月150時間 → 月60時間（60%削減） |
| 最適なソリューション | あなたの業界向け、中小企業特化 |
| お客様に寄り添う | 専任担当が3ヶ月伴走 |
| 課題解決 | スプレッドシート管理を卒業 |
| 革新的なアプローチ | API連携 + 自動メールの組み合わせ |
| シームレスな体験 | 既存ツールから1クリックでデータ移行 |
| ホリスティックなサービス | 採用 → 育成 → 定着 までを一画面で |
| 業界のリーディングカンパニー | 導入企業1,200社（自社調べ・2026年4月） |

## 「〜を実現します」回避

### NG
```
DX を実現します。
業務効率化を実現します。
成長を実現します。
```

### OK
```
スプレッドシート管理を卒業し、データ可視化へ。
月150時間の業務を、月60時間に減らします。
3ヶ月で「楽になった」と評価される業務へ。
```

## 主語を明確化

### NG
```
私たちは最適なサービスを提供します。
ユーザーの満足度を高めます。
```

### OK
```
[サービス名] は、中小企業の人事担当者向けに特化しています。
あなたの業務工数を、月150時間 → 月60時間に削減します。
```

## 競合置き換えテスト

書いたコピーを、競合社名に置き換えても成立しないか確認：

```
「私たちは業界をリードします」
→ どの会社でも書ける（NG）

「[サービス名] は、月150時間の採用業務を月60時間に削減します（自社調べ・100社平均）」
→ 自社固有のデータが含まれる（OK）
```

## チェックリスト

- [ ] 抽象語の連続使用（3つ以上）なし
- [ ] 競合置き換えテスト合格
- [ ] 業務の特定瞬間が描けている
- [ ] 担当者の本音が入っている
- [ ] 数字に出典付き
- [ ] 「〜を実現します」を多用していない
- [ ] 主語が明確
- [ ] 売り込みすぎていない
- [ ] CV後の安心感がある
