# Human Copy Rules

人間らしいコピーを書くための実践ルール。

## 6つの基本ルール

### ルール1: 抽象語を具体語に変える

```
❌ 業務効率を上げる
✅ 月150時間 → 月60時間（60%削減）

❌ 最適化する
✅ スプレッドシート管理を、ダッシュボード一画面に統合する

❌ 課題を解決する
✅ 「あの応募者、返信した？」を聞かれなくなる
```

### ルール2: 機能を利用シーンに変える

```
❌ API連携機能
✅ 求人媒体から応募者情報が自動で取り込まれます。
   毎週月曜の手作業のコピペは、もう不要です。

❌ ダッシュボード機能
✅ 朝、画面を開くだけで採用状況が把握できます。
   経営層への週次報告は、画面を共有するだけで完了します。
```

### ルール3: ベネフィットを業務・感情・評価に分解する

```
業務メリット: 月150時間の手作業 → 月60時間
感情メリット: 「これで失敗しないと思える」安心感
評価メリット: 上司から「楽になったね」と言われる
未来メリット: 来期、戦略採用に時間を使える
```

### ルール4: 読み手の言葉に寄せる

ペルソナの語彙・関心・心理に合わせる：

```
❌ パイプライン管理を最適化
   （マーケ用語、人事担当者には伝わらない）

✅ 応募者対応の状況が一覧で見える
   （人事担当者の業務語彙）
```

業界別の例：

- BtoB SaaS: 「業務工数」「ROI」「導入期間」
- 採用LP: 「成長機会」「カルチャー」「働き方」
- 医療: 「効果」「副作用」「使用方法」（薬機法に注意）
- 教育: 「身につく」「サポート」「卒業後」

### ルール5: 美辞麗句より現場感を優先する

```
❌ 業界をリードする革新的サービス
✅ 中小企業100社の採用業務を支援してきた経験から作りました

❌ お客様第一の姿勢で寄り添います
✅ 導入後3ヶ月、専任担当が毎週オンラインで伴走します
```

### ルール6: CTAは売り込みではなく次に進む理由にする

```
❌ お問い合わせはこちら
✅ 3分で資料を受け取る

❌ 今すぐお申し込みください！
✅ 14日間無料で試す（クレカ不要）

❌ 限定セール実施中
✅ 2026年X月X日まで、初期費用無料

❌ 詳細はこちら
✅ 同業種の事例を見てみる
```

## 各セクションのルール

### FV
- ターゲット明示「中小企業の人事担当者へ」
- ベネフィットを数値で「採用工数 60%削減」
- 信頼バッジを必ず（実績数値・認証・受賞）

### 課題提起
- 業務の特定瞬間を描く（「月曜の朝」「夜眠れない時」）
- 担当者の本音を入れる（「上司に怒られたくない」）
- 業界一般論を避ける

### 解決策
- ベネフィット → 機能 の順
- 機能の裏にあるロジックを説明
- 「だから、こう変わる」で締める

### 実績・事例
- 数値に必ず出典（「自社調べ・100社平均」）
- 同業種・同規模事例を優先
- お客様の声は許諾済みのみ

### FAQ
- 「ご安心ください」だけでは不十分
- 「共感 → 論拠 → 証拠」構造
- 不利情報も正直に

### 最終CTA
- ベネフィット再訴求
- 安心要素（解約・返金・営業電話の有無）
- CV後の流れ予告

## 美辞麗句リスト（避けるべき表現）

```
- 業界をリード
- お客様第一の姿勢
- 寄り添う
- 最適なソリューション
- 革新的なアプローチ
- ホリスティックなサービス
- シームレスな体験
- 最高品質
- 業界初
- 唯一無二
```

これらを使う場合は、必ず具体的な根拠と組み合わせる。

## 業界別「現場感」の出し方

### BtoB SaaS（採用管理）
- 「月曜の朝の応募者リスト」
- 「上司から『あの応募者、返信した？』」
- 「経営層への週次報告書」

### 採用LP（エンジニア）
- 「9:30の朝会で当日タスク確認」
- 「ペアプロでメンバーと相談」
- 「リモートでもオフィスでも自由」

### EC（化粧品）
- 「朝のスキンケア時間」
- 「メイクのノリが気になる日」
- 「肌に合うか試したい」

### 教育（プログラミングスクール）
- 「独学で挫折した過去」
- 「週末の2時間で学べる」
- 「ポートフォリオを作る不安」
