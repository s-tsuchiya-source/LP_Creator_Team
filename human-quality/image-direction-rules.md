# Image Direction Rules

LP画像・イラスト・ビジュアルのAIっぽさ排除ルール。

## 基本原則

```
画像は「装飾」ではなく「理解促進」のために使う。
意味のない画像は配置しない。
```

## NG パターン

### AI生成画像っぽい不自然さ
- 不自然な手・指
- 不自然な顔（左右非対称・違和感）
- 不自然な光沢・質感
- 不自然なディテール（背景の人物が溶けている等）

### 意味のない抽象画像
- 抽象未来都市
- 光の粒子・ニューラルネットワーク風の線
- 宇宙・銀河・抽象的な自然
- 商材と関係ない遠景

### 汎用的なストックフォト
- 「ビジネスマンの握手」
- 「ノートPCの上の手」
- 「黒板の前で説明する人物」
- 多人種・多国籍の働く人々

### 文脈に合わない画像
- BtoB なのに 抽象3Dイラスト
- 採用LP なのに ストックフォト
- 医療系 なのに ITイメージ

## OK パターン

### 商材の利用シーン

#### BtoB SaaS
- 実際の管理画面（モックアップ）
- ダッシュボードの視覚化
- ユーザーが業務している様子
- 業務フローの図解

#### 採用LP
- 実在社員の写真（許諾済）
- オフィス・働く環境
- メンバー同士の交流シーン
- 1日の流れの写真コラージュ

#### EC / 物販
- 実際の商品写真
- 利用シーン
- Before / After（誇大表現NG）
- 成分・素材の図解

#### コンサル / 高単価
- コンサルタントの実写真
- セミナー風景
- 過去支援企業のロゴ列
- 図解（フレームワーク等）

#### 採用LP（エンジニア）
- 実際のコードエディタ画面
- 技術スタック図
- ペアプロ風景（実在）
- リモートワーク環境

## 画像の優先順位

```
1. 実画面・実プロダクト（最高）
   └ SaaS 管理画面、EC 商品写真等

2. 実際の利用シーン
   └ 業務風景、社員写真

3. 図解・インフォグラフィック
   └ 業務フロー、Before/After

4. アイコン
   └ Material / Lucide 等

5. ストックフォト（最低限）
   └ 補助的に、文脈に合う場合のみ
```

## 業種別の画像方針

### BtoB（管理画面 / 業務フロー優先）

```
✅ 実際のダッシュボード画面
✅ 機能説明用のスクリーンショット
✅ 業務フロー図
✅ 比較表
❌ 抽象的なグラフィックのみ
❌ 関係ない人物写真
```

### 採用LP（職場の空気感優先）

```
✅ 実在社員の写真（許諾済）
✅ オフィスの様子
✅ メンバー同士の交流
✅ 1日の流れの写真
❌ ストックフォトのチームミーティング
❌ AI生成の人物イラスト
```

### EC / 物販（商品 + 利用シーン優先）

```
✅ 商品写真（複数アングル）
✅ 利用シーン
✅ Before / After（誇大NG）
✅ 成分・素材の図解
❌ モデルだけの写真
❌ 関係ない風景画像
```

### コンサル / 高単価（権威性優先）

```
✅ コンサルタント本人の写真
✅ 過去支援企業のロゴ列
✅ 著書・受賞の写真
✅ セミナー登壇の写真
❌ ストックフォトのスーツ姿
❌ 抽象的なビジネスイメージ
```

## AI生成画像を使う場合の注意

AI生成画像を使うことは可能だが、以下を守る：

### 必須チェック
- [ ] 不自然な手・指がない
- [ ] 不自然な顔がない
- [ ] 商材文脈に合う
- [ ] 装飾ではなく理解促進
- [ ] 著作権・利用規約に準拠

### 推奨されない用途
- FV のメインビジュアル
- 社員紹介の画像
- お客様の声の顔写真
- 実績・事例の画像

### 推奨される用途
- アイコン・補助イラスト
- 抽象的な概念図
- 装飾的な背景（控えめに）

## 画像の最適化

技術面での最適化（パフォーマンス）：

- 形式: WebP / AVIF + fallback
- サイズ: 適切な解像度（FV用 1920px / 補助 800px）
- 圧縮: 70-85% 品質
- レスポンシブ: srcset / sizes
- 遅延読み込み: loading="lazy"（FV以外）
- alt属性: 全画像に必須

## チェックリスト

- [ ] FV画像が商材文脈に合う
- [ ] 抽象的な装飾画像が過剰でない
- [ ] AI生成画像の不自然さがない
- [ ] 実画面・実利用シーン優先
- [ ] 業種別画像方針に沿う
- [ ] 全画像に alt属性
- [ ] WebP / AVIF 化
- [ ] 遅延読み込み（FV以外）
