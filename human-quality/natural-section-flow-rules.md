# Natural Section Flow Rules

セクション間の自然な流れを設計するルール。

## なぜ流れが重要か

LPは「文章の集合」ではなく「物語」です。
セクションが断絶していると、読者の感情が途切れて離脱します。

## 自然な流れの基本

```
[1. 興味] → [2. 共感] → [3. 期待] → [4. 信頼]
       ↓
[5. 不安解消] → [6. 行動意欲] → [7. 決断] → [8. 安心]
```

各段階で前の段階を受け、次の段階へ橋渡しする。

## セクション間の橋渡し

### FV → 課題提起

```markdown
## FV
中小企業の人事担当者へ
採用業務を最大60%削減する [サービス名]

[CTA]

---

## 課題提起
「業務工数を削減したい」という気持ちは、
多くの人事担当者が共有しています。

具体的には、こんな状況ではありませんか？
□ ...
```

→ FVの「採用業務削減」を受けて、課題セクションで具体化。

### 課題提起 → 解決策

```markdown
## 課題提起
□ 月曜の朝、応募者リストを見るのが憂鬱
□ 上司から「あの応募者、返信した？」と聞かれてヒヤヒヤ
...

ひとつでも当てはまったら、続きをお読みください。

---

## 解決策
[サービス名] が、これらの悩みを解決します。

採用業務を一元管理し、求職者対応を自動化することで、
採用工数を最大60%削減します（自社調べ・100社平均）。
```

→ 課題提起で挙げた悩みを、解決策で受ける。

### 解決策 → 機能 / 強み

```markdown
## 解決策
採用工数を最大60%削減します。

---

## なぜ実現できるのか
[サービス名] は、3つの「無駄」を排除します：

1. 二重入力の無駄
2. 対応漏れの無駄
3. 報告書作成の無駄
```

→ 「なぜ」で繋げる。

### 機能 → 実績 / 事例

```markdown
## 機能
3つの自動化機能を提供します。

---

## 実証された効果
すでに1,200社（自社調べ）で、
平均60%の工数削減を実現しています。
```

→ 「すでに」で繋げる。

### 実績 → 比較表

```markdown
## 実績
1,200社が選ぶ理由は、業界トップクラスの中小企業特化設計です。

---

## 他社との違い
具体的に、何が違うのか。
比較表でご確認ください。
```

→ 「具体的に」で繋げる。

### 比較 → 価格

```markdown
## 比較表
中小企業向けの価格帯と、専任担当による導入支援。

---

## 価格プラン
中小企業に最適化した3プランをご用意しました。
```

→ 「最適化」で繋げる。

### 価格 → FAQ

```markdown
## 価格プラン
- Light: ¥9,800/月
- Standard: ¥29,800/月（推奨）
- Pro: ¥98,000/月

---

## よくあるご質問
「月額9,800円は、中小企業にとって高くないですか？」

ROI試算では... [回答]
```

→ 価格セクションで生まれる疑問を、FAQで解消。

### FAQ → 最終CTA

```markdown
## FAQ
解約自由 / 無料トライアル / 営業電話なし

---

## まずは資料を見てみませんか？
3分で送られる資料には、以下が含まれます：

✓ 機能一覧と業務適用例
✓ 同業種の導入事例 5社
...

[3分で資料を受け取る]
```

→ FAQで安心したら、CTAでハードル低い行動へ。

## 流れを断絶させる NG パターン

### NG型1: 唐突な機能紹介

```
## FV
ターゲット明示 + ベネフィット

## 機能
- API連携
- 自動メール
- ダッシュボード

→ 課題共感なしに機能紹介で離脱
```

### NG型2: 解決策セクションがない

```
## 課題提起
□ ...

## 実績
1,200社が選んでいます

→ 何を実現するLPか不明
```

### NG型3: 比較が浮く

```
## 解決策

## 比較表
（前後脈絡なし）

## 機能

→ 比較表が孤立
```

### NG型4: FAQ が CTAから遠い

```
## FAQ

## 価格プラン
（FAQでの不安解消が活きない）

## 最終CTA
```

→ FAQ後すぐにCTAが原則。

## セクション順序の推奨

```
1. FV（流入意図受け止め + 興味）
2. 課題提起（共感）
3. 解決策（期待）
4. 機能 / 強み（期待強化）
5. 実績 / 事例（信頼）
6. 比較（納得）
7. 価格 / プラン
8. 導入フロー
9. FAQ（不安解消）
10. 最終CTA（行動意欲・決断）
```

## 業種別の順序差分

### 採用LP
```
FV → 「こんな方を求めています」 → 仕事内容 → 1日の流れ → 社員インタビュー → 制度 → 選考フロー → FAQ → CTA
```

### EC / 物販
```
FV → 商品特徴 → Before/After → 成分 → お客様の声 → 価格 → キャンペーン → FAQ → CTA
```

### コンサル / 高単価
```
FV（権威） → 課題 → アプローチ → コンサル紹介 → 実績 → 事例 → 料金 → FAQ → CTA
```

## チェックリスト

- [ ] セクション間の橋渡しがある
- [ ] 流れが断絶していない
- [ ] 課題提起 → 解決策の繋がり
- [ ] 機能 → 実績 → 比較 の論理
- [ ] FAQ が最終CTA直前
- [ ] 業種別の順序差分を反映
