# industry-playbooks/ — 業種別LP攻略書

industry-playbooks は、業種別のLP制作ノウハウ集です。

## 役割

- 業種特性を踏まえた訴求軸
- 意思決定者の特性
- 業界別の刺さるオファー
- 業界別の法務リスク
- 業界別の信頼要素

## ファイル一覧

| Playbook | 用途 |
|---|---|
| btob-saas-playbook.md | BtoB SaaS |
| hr-recruiting-playbook.md | 採用LP |
| hr-ads-playbook.md | 求人広告連動LP |
| consulting-playbook.md | コンサルティング |
| school-education-playbook.md | スクール / 教育 |
| seminar-webinar-playbook.md | セミナー / ウェビナー |
| ec-product-playbook.md | EC / 物販 |
| local-service-playbook.md | ローカルサービス |
| high-ticket-service-playbook.md | 高単価サービス |

## 使い方

各エージェントは、案件の業種が判別された段階で該当 playbook を参照する。`lp-director` は、Brief → Strategy フェーズで playbook を選定し、後段エージェントに渡す。
