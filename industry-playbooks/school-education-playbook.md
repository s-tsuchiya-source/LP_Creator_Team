# School / Education LP Playbook

## Target Market

スクール / 教育 / プログラミングスクール / 資格取得 / 子供向け教育

## Common User Pain

- スキル不足で転職できない
- 独学で挫折
- 業界の最新動向についていけない
- 資格を取って昇進したい

## Decision Makers

- **本人**（受講者）
- **配偶者**（家計判断・社会人スクール）
- **保護者**（子供向けスクール）

## Conversion Goals

- 主CV: 体験授業申込 / 資料請求 / 説明会予約
- 補助CV: カリキュラム閲覧 / 講師紹介閲覧

## Winning Offers

- 無料体験授業
- 資料請求（カリキュラム / 卒業生事例）
- 個別カウンセリング
- 期間限定割引（実態に即す）

## High-Converting Structure

1. FV（変化訴求型 + 主CTA）
2. 「こんな悩み、ありませんか」
3. 講座で何ができるようになるか
4. カリキュラム（詳細）
5. 講師紹介
6. 受講生の声 / Before-After
7. サポート体制
8. 料金プラン（分割払い表示）
9. 受講までの流れ
10. FAQ
11. 最終CTA

## Proof Elements

- 講師経歴
- 卒業生実績（就職先 / 資格取得率 / 転職成功率）
- 受講生の声（顔写真 + 実名 / 許諾済）
- メディア掲載・受賞
- 認定機関

## Common Objections

- 「料金が高い」→ 分割払い / 教育訓練給付金
- 「本当に身につくか」→ 卒業生事例 / 返金保証
- 「途中で挫折しないか」→ サポート体制 / メンター
- 「卒業後の支援」→ 就職サポート

## Copy Angles

- 変化訴求（Before / After）
- 講師の権威性
- 卒業生の成功事例
- サポート体制の手厚さ

## Design Direction

- 親しみやすさ + 信頼感
- 写真は実講師・実受講生
- 卒業生Before/Afterを視覚的に
- 料金は明確に（分割表示も）

## Measurement Points

- 主CV: form_submit (trial_lesson / resource_request / consultation)
- 補助CV: video_play / pdf_download
- ファネル: page_view → カリキュラム閲覧 → CTA → form_submit

## Legal / Risk Notes

- 特定商取引法表示
- 割賦販売法（分割払い時）
- 「絶対に転職できる」NG
- 「100%資格取得」NG
- 受講生事例に「個人差」明記
- 返金保証は実行可能範囲
