# knowledge/ — LP制作ナレッジベース

LP制作で繰り返し参照する知識を集約。各エージェントは必要に応じてここを参照する。

## ディレクトリ構成

```
knowledge/
├─ lp-patterns/              ← 業界別LPパターン
├─ copy-frameworks/          ← コピーライティングフレームワーク
├─ section-patterns/         ← セクション別パターン
├─ design-patterns/          ← デザインパターン
├─ measurement-patterns/     ← 計測パターン
└─ legal-expression-rules/   ← 法務表現ルール
```

## 利用ルール

- 関連するパスのみ参照（全読み禁止）
- 矛盾があれば既存ファイルを優先し、追記時はユーザー確認
- 知見追加時は重複を避け、既存ファイルを更新優先
- 機密情報・顧客固有情報は記載しない（公開リポジトリ前提）
