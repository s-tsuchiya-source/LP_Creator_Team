# AIDA フレームワーク

## 構造
- **A**ttention: 注意を引く
- **I**nterest: 興味を引く
- **D**esire: 欲求を高める
- **A**ction: 行動を促す

## LPでの使い方

| セクション | AIDA要素 |
|---|---|
| FV | Attention |
| 問題提起・解決策 | Interest |
| 強み・実績・事例 | Desire |
| CTA | Action |

## 適合する案件
- 認知度の低いサービス
- 新規カテゴリ商材
- 興味喚起から始める必要がある場合

## 注意点
- AIDA は古典的だが、現代のLPでは PASONA / QUEST のほうが具体的
- AIDA を骨子に、PASONA / QUEST 要素を補完するのが現実的
