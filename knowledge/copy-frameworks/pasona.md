# PASONA フレームワーク

## 構造
- **P**roblem: 問題提起
- **A**ffinity: 親近感（共感）
- **S**olution: 解決策
- **O**ffer: 提案（オファー）
- **N**arrowing down: 絞り込み（限定性）
- **A**ction: 行動（CTA）

## LPでの使い方

| セクション | PASONA要素 |
|---|---|
| 問題提起セクション | Problem |
| 共感セクション | Affinity |
| 解決策セクション | Solution |
| 商品 / オファー | Offer |
| 限定性・緊急性 | Narrowing down |
| CTA | Action |

## BtoB向けの型
- P: 「[業務] に[時間 / コスト] がかかっていませんか？」
- A: 「同じ悩みを抱える企業様から多数のご相談をいただいています」
- S: 「[プロダクト名] が、[業務] を[改善幅] します」
- O: 「初期費用無料・最短[期間] で導入可能」
- N: 「先着[数] 社限定」「期間限定[特典]」
- A: 「無料相談を予約する」

## 採用LP向けの型
- P: 「今の仕事に物足りなさを感じていませんか？」
- A: 「同じように悩み、当社で活躍する社員が多くいます」
- S: 「[ミッション] に挑む環境で、あなたの[強み] を活かせます」
- O: 「カジュアル面談から、[特徴的な制度]」
- N: 「[期間] の説明会開催」
- A: 「カジュアル面談を予約する」

## 使用時の注意
- Problem を煽りすぎない（不安商法と紙一重）
- Affinity に嘘がないように
- Narrowing down を常時セールにしない（景表法）
