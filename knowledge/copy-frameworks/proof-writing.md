# 実績・証拠の書き方

## 信頼性訴求の優先順位

1. **数値実績**（具体的・出典付き）
2. **第三者評価**（受賞・認証・メディア掲載）
3. **顧客の声**（写真・実名・許諾済）
4. **導入企業ロゴ**（許諾済）
5. **専門家のコメント**

## 数値実績の書き方

### NG
- 「導入実績多数」
- 「業界No.1」（根拠なし）
- 「最高品質」

### OK
- 「導入企業 XXX 社（YYYY年MM月時点）」
- 「[調査会社] 調べによる[カテゴリ]シェアNo.1（YYYY年実績）」
- 「[認証] 取得済み」

## お客様の声の書き方

### 必須要素
- 顔写真 or イラスト（許諾済）
- 氏名 or イニシャル + 業種・役職
- 課題 → 効果のストーリー
- 具体的数値（あれば）

### NG
- 「すごく効果がありました！」だけ
- イニシャルなしでも特定できる情報
- 写真の無断使用

## 比較表の書き方

### NG
- 自社=○ / 競合=× の二元論
- 競合名を直接記載（誹謗中傷リスク）

### OK
- 自社=○ / A社=○ / B社=△ の段階評価
- 競合名はA社・B社などで匿名化（または許諾済）
- 各項目に「機能」「使いやすさ」「価格」「サポート」など客観的評価軸

## メディア掲載の見せ方

- 媒体ロゴ + 掲載日 + リンク
- リンク切れに注意
- 紙媒体は掲載号 + ページ
