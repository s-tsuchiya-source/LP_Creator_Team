# QUEST フレームワーク

## 構造
- **Q**ualify: 読者を絞り込む
- **U**nderstand: 共感する
- **E**ducate: 教育する
- **S**timulate: 興奮を引き出す
- **T**ransition: 行動への移行

## LPでの使い方

| セクション | QUEST要素 |
|---|---|
| FV | Qualify「あなたは〜ではないですか？」 |
| 共感 | Understand「私もそうでした」 |
| 解決策・機能 | Educate「[手法] で解決できます」 |
| 実績・事例 | Stimulate「[結果] を実現した方も」 |
| CTA | Transition「今すぐ[行動] へ」 |

## 強み
- 読者を絞り込んで密度の濃い訴求が可能
- ベネフィットを段階的に積み上げ
- 不安解消に強い

## 適合する案件
- BtoB高単価
- 教育・スクール
- コンサルティング
- 高関与購買商品
