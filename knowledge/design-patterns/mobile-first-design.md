# モバイルファーストデザイン

## 想定
- LP流入の70〜80%がスマホ
- スマホで価値が伝わらないとCV以前に離脱

## 必須要件
- フォントサイズ 16px 以上（ズーム回避）
- タップターゲット 44x44px 以上
- 横スクロール発生なし
- フォームは1カラム
- 追従CTA（フッター固定）
- ハンバーガーメニュー検討

## ブレークポイント設計

```css
/* SP first */
.section { ... }

/* Tablet */
@media (min-width: 640px) { ... }

/* PC */
@media (min-width: 1024px) { ... }
```

## SP特有の設計
- 1カラムレイアウト
- 縦スクロールが基本
- 重要情報は折り畳まない
- 画像は縦長 or 正方形
- 動画はサイレント自動再生（タップで音声）

## 単位設計
- フォント: rem
- レイアウト: % / vw / clamp()
- 単位を媒体間で混在させない（境界で段差発生原因）

## 追従CTA
- フッター固定（position: fixed; bottom: 0）
- スクロール時のみ表示（FV では非表示）
- フォーム遷移時は非表示
- 高さ: 60〜72px
