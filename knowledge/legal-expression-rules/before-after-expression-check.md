# Before/After 表現チェック

## 該当法令
- 景品表示法
- 薬機法（化粧品・健康食品）

## NG表現
- 加工 / 修正された画像
- 異なる条件下での比較（光・角度）
- 個人差を一切示さない断定
- 効果効能の医薬品的表示

## OK 要件

### 写真・画像
- 加工なし or 同条件で撮影
- 個人差表記必須（※個人の感想です / 個人差があります）
- 撮影時期・条件の明示

### 数値Before/After
- 出典・調査方法の明示
- 「平均値」「中央値」の明示
- サンプル数の明示

## 例

### NG
```
Before: シミだらけの肌
After: つるつるの肌
※ 効果には個人差があります  ← 「効果」と書くと薬機法該当
```

### OK
```
Before: スキンケアに月10時間
After: 月3時間（推定）

※ 当社調べ・対象者100名・2026年3月実績
```

### 推奨
- 「効果」と表現せず「使用前 / 使用後」「導入前 / 導入後」と中立に
- 視覚的な変化はターゲットへの暗示効果が高いため、薬機法に注意
- 化粧品なら「整える」「うるおいを与える」の範囲

## サービス系の Before / After
- 業務効率（時間 / コスト）の Before / After は安全
- ただし「導入したら必ず X 削減」と断定はNG
