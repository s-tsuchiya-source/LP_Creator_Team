# 根拠が必要な表現一覧

## 必ず根拠が必要

| 表現カテゴリ | 例 | 必要な根拠 |
|---|---|---|
| 最大級表現 | 「No.1」「最高」「最安」 | 第三者調査・調査範囲・時期 |
| 数値実績 | 「導入XXX社」「XX%削減」 | 調査主体・条件・サンプル数 |
| 効果断定 | 「絶対」「100%」「必ず」 | 通常NG（避けるのが安全） |
| 比較表現 | 「他社比2倍」 | 比較対象・比較条件 |
| 受賞・認証 | 「[賞] 受賞」 | 賞の正式名称・受賞年・主催 |
| 専門家推薦 | 「医師推奨」 | 推薦者氏名・所属・推薦内容 |

## 業界別の必要表記

### 化粧品・健康食品（薬機法）
- 効果効能は化粧品の範囲内（56項目）
- 「個人の感想です」明記
- 表現監修者が望ましい

### 医療系（薬機法）
- 医薬品の効能効果に該当する表現NG
- 健康食品で「治る」「効く」NG
- 治験結果はガイドラインに沿う

### 金融（金商法）
- 「絶対儲かる」NG
- リスクの明示
- 想定利回りの根拠

### 不動産（宅建業法）
- 価格・物件情報の正確性
- 誇大広告NG（断定的表現）
- 取引条件の明示

### 教育・スクール（特商法・割販法）
- 「絶対に転職できる」NG
- 受講生の体験談に「個人差」明記
- 分割払いの総額・金利明示

## 出典フォーマット

```
[実績数値] （[調査主体] 調べ / [対象] / [時期]）
```

例:
- 「導入企業 1,200社（自社調べ・2026年4月時点）」
- 「顧客満足度 95%（[調査会社] 調べ・対象者300名・2025年12月）」

## 検証手順
1. 表現抽出
2. 必要根拠の有無確認
3. 出典明記
4. 不足時は修正案提示
