# 導入事例セクション

## 役割
- 具体的な成功ストーリー
- ターゲット類似事例で「自分にも」を喚起

## 構成
- 事例カード 3〜5件
- 各カードに以下を含める：
  - 企業ロゴ / 担当者写真
  - 業種・規模
  - 課題
  - 導入の決め手
  - 効果（数値）
  - 担当者コメント

## 推奨

- 同業種・同規模の事例を優先
- 数値効果を必ず含める
- リンクで詳細事例ページに誘導

## 法務
- 顧客の許諾必須
- NDA該当情報は出さない
- 数値の検証可能性を担保

## 例

```
事例: [株式会社○○]
- 業種: SaaS
- 規模: 従業員200名
- 課題: 採用業務の属人化
- 導入の決め手: 既存ツールとのAPI連携
- 効果: 採用工数 月150時間 → 月60時間（60%削減）
- 担当者コメント: 「導入後3ヶ月で、採用業務が見える化されました」
```
