# 最終CTAセクション

## 役割
- 最後の決断を促す
- ベネフィット再訴求
- 限定性・緊急性で行動喚起

## 構成
- 見出し（再訴求）
- ベネフィット箇条書き 3個
- 限定性 / 緊急性
- メインCTA + サブCTA
- 安心要素（保証・解約条件 等）

## 例

```
[サービス名] で、採用業務を変えませんか？

✓ 採用工数を最大50%削減
✓ 応募者対応の質を向上
✓ 経営層への報告が自動化

導入企業 1,200社が選ぶ採用管理ツール

[今すぐ無料で資料請求] [無料デモを予約]

★ 14日間無料トライアル / クレジットカード不要
★ いつでも解約可能
```

## 注意点
- ベネフィットはFVと整合
- 限定性は嘘でない範囲で
- 安心要素は反論処理の最終仕上げ
- CTAは2種類用意（メイン + サブ）
- 主CTAは最も低負荷な行動（資料請求 / 無料相談）
