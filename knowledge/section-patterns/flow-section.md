# 導入フローセクション

## 役割
- 行動後の流れを可視化
- 不安解消（「申し込んだら何が起きる？」）

## 構成
- 4〜5ステップ
- 各ステップにアイコン・所要時間
- 図解形式

## 例（BtoB SaaS）

```
1. 資料請求（30秒）
   フォーム入力 → 即座にメールで資料受領

2. 個別相談（30〜60分）
   オンラインで貴社課題をヒアリング

3. デモ・トライアル開始（1〜2週間）
   無料で実環境で試用

4. 契約・本番導入（1ヶ月）
   契約 → 設定 → スタッフ研修

5. 運用開始
   専任担当が継続サポート
```

## 注意点
- 各ステップに具体的所要時間
- 不安箇所には補足（「個人情報は◯◯」等）
- 資料請求後の流れも明示
