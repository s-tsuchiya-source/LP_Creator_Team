# LP Structure Blueprint

LP Structure Blueprint は、LPとして必ず押さえるべき構成の流れを定義する設計基盤です。

## Purpose

最高品質のLPは、単に情報を並べるのではなく、読み手の心理が自然にCVへ進む順序で構成される必要があります。

## Core Flow

1. 流入意図との一致
2. ファーストビューで自分向けだと分かる
3. 課題・悩みを言語化する
4. 共感と放置リスクを示す
5. 解決策を提示する
6. なぜ解決できるかを説明する
7. 短期・中期・長期メリットを示す
8. 証拠・実績・事例で信頼を作る
9. 競合・代替手段との違いを示す
10. オファーを提示する
11. 不安・反論を解消する
12. CV後・導入後の流れを見せる
13. CTA直前の行動理由を作る
14. 最終CTAで背中を押す
15. CV後の安心を伝える

## Rule

案件タイプにより省略できるセクションはありますが、`lp-director` は省略理由を必ず記録します。
