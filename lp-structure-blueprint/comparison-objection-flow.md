# Comparison & Objection Flow

競合・代替手段との比較と、不安解消FAQ の流れ。

## 役割

- 「これを選ぶ理由が分かる」納得感を作る
- 6カテゴリ不安を解消し、CV直前の心理ハードルを下げる
- 「契約後も自由」と感じる安心設計

## 必須要素

### 比較セクション
- 比較対象（競合 / 既存代替手段）
- 客観的評価軸
- 自社の優位点 + 劣位点（信頼性）

### FAQ
- 6カテゴリ不安への対応
- 5〜10件
- 「共感 → 論拠 → 証拠」構造

## 良い例

### 比較表（型A: 競合との比較）

```markdown
## 他社との違い

| 項目 | 自社 | A社 | B社 |
|---|---|---|---|
| 主要機能 | ◎ | ○ | △ |
| 中小企業対応 | ◎ | △ | ○ |
| 月額（初年度） | ¥9,800〜 | ¥50,000〜 | ¥29,800〜 |
| 導入期間 | 最短2週間 | 1〜2ヶ月 | 1ヶ月 |
| サポート | チャット即応 | メールのみ | メールのみ |

※ 各社公式情報および公開資料（2026年4月時点）
```

### 比較表（型B: 既存代替手段との比較）

```markdown
## こんな方法、もう不要です

| 項目 | スプレッドシート | 人力管理 | [自社] |
|---|---|---|---|
| 工数 | 多い | 多い | 少ない |
| 抜け漏れ | あり | あり | なし |
| 経営報告 | 手作業 | 手作業 | 自動 |
| 拡張性 | 低 | - | 高 |
```

### FAQ（6カテゴリ網羅）

```markdown
## よくあるご質問

**Q. 月額9,800円は、中小企業にとって高くないですか？（価格不安）**

A. ROI試算では、月50時間の工数削減で約15万円相当の効果。
   月額の約15倍のリターンが期待できます。
   詳細は、無料の試算資料でご確認ください。

**Q. 導入時間はどれくらいかかりますか？（導入不安）**

A. 最短2週間です。専任担当が支援します。
   データ移行が必要な場合も、無料で代行可能です。

**Q. 効果は本当に出ますか？（効果不安）**

A. 同業種の導入事例（A社、B社、C社）で、
   平均60%の工数削減実績があります（自社調べ）。
   ただし効果は個人差・業種差があります。

**Q. 使いこなせるか不安です（運用負荷不安）**

A. 多くの方が同じご不安をお持ちです。
   研修・チャットサポート・FAQ動画を完備。
   導入企業の85%が「想定より簡単」と回答しています。

**Q. 上司への説明資料はありますか？（社内説明不安）**

A. 社内検討用の資料を、無料でご用意しています：
   - ROI試算シート
   - 競合比較表
   - 同業種事例集
   - セキュリティチェックリスト

**Q. 解約は自由にできますか？（失敗不安）**

A. 月単位で解約可能です。最低利用期間はありません。
   データはいつでもCSVエクスポート可能です。
```

## 悪い例

### NG型1: 自社全項目 ○ / 競合全項目 ×

```
| 項目 | 自社 | A社 |
| 機能 | ◎ | × |
| 価格 | ◎ | × |
| サポート | ◎ | × |
```

→ 信頼性ゼロ。誇大表現。

### NG型2: 評価軸が主観的

```
| 項目 | 自社 | A社 |
| 使いやすさ | ◎ | △ |
| カッコよさ | ◎ | × |
```

→ 客観性なし。

### NG型3: 「ご安心ください」だけのFAQ

```
Q. 効果はありますか？
A. はい、ご安心ください。最高の効果をお約束します。
```

→ 具体性なし。法務リスク。

### NG型4: 不利情報の隠蔽

```
（解約条件・最低利用期間を一切書かない）
（返金条件を曖昧にする）
```

→ 後でトラブル。

## 6カテゴリ不安への対応

### 1. 価格不安 → ROI / 分割の期待
- ROI試算
- 分割払い
- 14日間無料トライアル

### 2. 導入不安 → 安心の期待
- 導入フロー図
- 専任担当紹介
- 同業種の導入事例

### 3. 効果不安 → 確信の期待
- 同業種・同規模事例
- 数値実績（出典付き）
- 効果保証 / 返金保証（範囲内）

### 4. 運用負荷不安 → サポートの期待
- サポート体制
- 教育コンテンツ
- 伴走支援
- 現場定着率データ

### 5. 社内説明不安 → 推進力の期待
- 社内検討資料
- ROI試算
- 比較表
- セキュリティ資料

### 6. 失敗不安 → 安全の期待
- 解約条件の明示
- データポータビリティ
- 無料トライアル
- 返金保証（範囲内）
