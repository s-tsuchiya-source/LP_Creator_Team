# Offer & CTA Flow

オファー設計と CTA フローの設計。

## 役割

- 「今動く理由」を作る
- ハードル低い行動から始められるオファー設計
- CTA 直前に行動意欲を最大化
- CV 後の流れを予告し、安心感を提供

## 必須要素

### オファー
- メインオファー（資料請求 / 無料相談 / 14日間無料トライアル）
- サブオファー（デモ / 個別相談）
- 限定性（実態に即す）
- 保証（実行可能範囲）

### CTA
- ハードル段階別（最低2個）
- 「動詞 + ベネフィット」文言
- 安心要素（解約・返金・営業電話の有無）
- CV後の流れ予告

## 良い例

### 型A: BtoB SaaS（ハードル段階別）

```markdown
## まずは資料を見てみませんか？

導入企業1,200社が選ぶ採用管理ツール、
3分で送られる資料には、以下が含まれます：

✓ 機能一覧と業務適用例
✓ 同業種の導入事例 5社
✓ 価格・プラン詳細
✓ ROI試算シート
✓ セキュリティ情報

[3分で資料を受け取る]   [無料デモを予約]

★ 14日間無料トライアル / クレジットカード不要
★ いつでも解約可能
★ 専任担当による導入支援

### 資料請求後の流れ

1. メールで資料が届きます（即時）
2. 担当者からのご挨拶メール（3営業日以内）
3. ご希望の方のみ、個別相談を実施
4. 営業からの強引な連絡はありません
```

### 型B: 採用LP（ハードル超低）

```markdown
## あなたの挑戦を、待っています

✓ モダンな技術スタック × 自由な働き方
✓ 急成長中のSaaSで、裁量ある仕事
✓ 個人の成長を全力支援する制度

カジュアル面談から、気軽に始めませんか？

[カジュアル面談を予約する]   [メンバーと話す]

★ 業務内容・カルチャーを気軽にお聞きください
★ 応募ではないので、お断りも歓迎
★ 履歴書不要・事前準備不要
```

### 型C: 高単価コンサル（権威性 + 安心）

```markdown
## 30分の無料相談から始めませんか

経営者・事業責任者向けに、貴社の課題に合わせた
個別アドバイスを行います。

✓ 業界別事例集をプレゼント
✓ ROI試算シートを無料提供
✓ 強引な営業は一切ありません

[30分無料相談を予約]   [資料請求]

★ 30分・オンライン
★ お申込後、こちらから日程候補をご連絡
★ ご都合に合わせて柔軟に対応
```

## 悪い例

### NG型1: CTA1つだけ

```
[お問い合わせ]
```

→ ハードル選択肢なし。離脱率高。

### NG型2: 「お問い合わせ」だけ

```
[お問い合わせ]
[詳細はこちら]
```

→ ベネフィット欠如。CV理由不明。

### NG型3: 嘘の限定性

```
本日限定！
今だけ特別価格！
※ 実際は常時セール
```

→ 景表法違反。信頼毀損。

### NG型4: CV後の流れ不明

```
[今すぐ申し込む]
（申込後の流れの説明なし）
```

→ 不安で離脱。

### NG型5: 営業されすぎ不安への対応なし

```
[資料請求]
（営業電話の有無に触れない）
```

→ 「営業されすぎないか」不安で離脱。

## CTA 文言の例

### ハードル低い
- 「3分で資料を受け取る」
- 「メールアドレスだけで資料DL」
- 「無料で診断する」
- 「カジュアル面談を予約する」

### ハードル中
- 「無料デモを予約する」
- 「個別相談（無料）に申し込む」
- 「14日間無料で試す（クレカ不要）」

### ハードル高
- 「契約手続きへ進む」
- 「今すぐ申し込む」
- 「購入する」

## 安心要素テキスト

```
★ クレジットカード不要
★ いつでも解約可能
★ 営業からの強引な連絡なし
★ 個別相談はご希望の方のみ
★ メルマガ登録は任意
★ 解約はメール1通でOK
```

## CV 後の流れ予告

CV 直前の最終 CTA セクションに、以下を必ず明示：

```markdown
### 申込後の流れ

1. **すぐに**: メールで資料が届きます
2. **3営業日以内**: 担当者からご挨拶メール
3. **任意で**: 個別相談（30分・オンライン）
4. **無料で**: 14日間トライアル（クレカ不要・自由解約）
```

## 限定性の作り方

### NG（嘘の限定性）
- 「本日限定」「今だけ」（常時セール）
- 「先着10名」（実際は無制限）

### OK（実態に即す）
- 「2026年X月X日まで」（実際の期日）
- 「先着50社限定」（実際の制限）
- 「初年度のみ無料」（実際の条件）

## Validation 観点

- [ ] メイン + サブオファー
- [ ] CTA 最低2個（ハードル段階別）
- [ ] CTA 文言が「動詞 + ベネフィット」
- [ ] 安心要素 3つ以上
- [ ] CV 後の流れ予告
- [ ] 限定性が実態に即す
- [ ] 営業電話の有無を明示
