# Post CV Experience Flow

CV 後の体験設計。サンクスページ・自動返信メール・次のステップで、
CV ユーザーの不安を解消し、商談化率を高める。

## 役割

- 「申込後何が起きるか」の不安を解消
- CV 後のステップを明示
- 関連資料への誘導で温度感維持
- 営業電話・メルマガの方針を明示

## 必須要素

- サンクスページ
- 自動返信メール
- 次のステップ提示
- 営業対応方針
- 関連資料・SNSへの誘導
- 計測タグ（CV発火）

## 良い例

### サンクスページ（BtoB SaaS）

```markdown
# 資料請求ありがとうございます

ご記入いただいたメールアドレスに、
資料を送信いたしました。

## 次の流れ

### すぐに
- ✉️ 資料が届きます
- （迷惑メールフォルダもご確認ください）

### 3営業日以内
- 担当者からご挨拶メールをお送りします
- 営業電話はかけません

### ご希望時
- 個別相談（30分・オンライン）
- 無料デモ
- 14日間無料トライアル

## よくあるご質問

**Q. 資料が届かない場合は？**
迷惑メールフォルダをご確認ください。
3時間以上経っても届かない場合は、こちらまで：[メールアドレス]

**Q. 営業電話はありますか？**
かけません。
ご希望の方のみ、メールで個別相談をご案内します。

**Q. デモを予約したいのですが？**
こちらから24時間予約可能：[予約URL]

## 関連資料

- [競合比較表 PDF]
- [ROI試算シート]
- [導入事例集]

## SNS でも情報発信中

- Twitter: [@xxx]
- LinkedIn: [URL]
```

### サンクスページ（採用LP）

```markdown
# カジュアル面談のご予約、ありがとうございます

## ご予約内容

- 日時: [予約日時]
- 形式: オンライン
- 担当: [担当者名]

## 当日までの流れ

### 1〜2営業日以内
担当者よりご確認メールをお送りします。
ZoomのURLもお送りします。

### 当日（30分）
- 自己紹介・お話のきっかけ
- 仕事内容・カルチャーの説明
- ご質問にお答えします
- カジュアルな雑談形式です

### 後日
- 興味があれば、エントリーへ
- 興味がなくても、お断りでOKです

## ご準備不要です

- 履歴書不要
- 事前資料不要
- 服装自由（オンラインなのでカジュアルでOK）

## ご質問・キャンセル

[連絡先メールアドレス]

## 採用情報

- 採用ピッチ動画: [URL]
- 社員インタビュー: [URL]
- メンバー紹介: [URL]
```

```
このメールにご返信ください。

## 営業電話について

営業電話はかけません。

ご希望の方のみ、メールで個別相談をご案内します。

## 14日間 無料トライアル

クレジットカード不要で、すぐに開始できます：
[トライアル開始URL]

何卒よろしくお願いいたします。

[サービス名] 担当 [氏名]
[会社名]
[連絡先メールアドレス]
```

## 悪い例

### NG型1: サンクスページが汎用的すぎる

```
お申し込みありがとうございます。
担当者よりご連絡いたします。
```

→ 不安が残る。

### NG型2: 自動返信メールなし

```
（送信完了画面のみ表示、メールなし）
```

→ 「本当に届いた？」と不安。

### NG型3: 営業電話の有無が不明

```
（営業電話があるのか、ないのか書かない）
```

→ 不安で離脱。

### NG型4: 計測タグなし

```
（CV計測タグが発火しない）
```

→ 広告効果測定不能。

## 計測タグ（必須）

サンクスページで以下を発火：

```javascript
// GA4 CV
gtag('event', 'form_submit', {
  form_id: 'main_form'
});

// Google Ads CV
gtag('event', 'conversion', {
  send_to: 'AW-XXXXXXXXX/YYYYYYYYY'
});

// Meta Pixel
fbq('track', 'Lead');

// Enhanced Conversion（メール・電話番号ハッシュ化）
gtag('set', 'user_data', {
  email_address: hashedEmail,
  phone_number: hashedPhone
});
```

## URL パラメータ チェック（重複計測防止）

サンクスページが直接アクセスでも到達できる場合：

```javascript
// CV パラメータがある場合のみ計測
const params = new URLSearchParams(window.location.search);
if (params.get('cv') === 'success') {
  // 計測タグ発火
}
```

## Validation 観点

- [ ] サンクスページ存在

- [ ] 自動返信メール内容設計
- [ ] CV 計測タグ発火

- [ ] 関連資料・SNS への誘導
