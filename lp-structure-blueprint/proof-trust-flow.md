# Proof & Trust Flow

実績・事例・数字・第三者評価で信頼性を担保する流れ。

## 役割

- 「実在の会社が、実在の成果」を証明
- 「失敗しないと思える」確信を作る
- 同業種事例で「自分にもできる」と思わせる
- 法的に問題のない数値表現で信頼性を保つ

## 必須要素（最低3種類配置）

- 数値実績（出典付き）
- 導入企業ロゴ列（許諾済）
- 受賞・認証
- お客様の声（許諾済）
- メディア掲載
- 第三者調査・専門家推薦

## 良い例

### 型A: 数値 + 導入企業 + お客様の声

```markdown
## 多くの企業が選ぶ理由

### 数値実績
- 導入企業 1,200社（自社調べ・2026年4月時点）
- 平均60%の業務工数削減（100社平均・2025年実績）
- 顧客継続率 92%（自社調べ・2025年）

※ 効果には個人差・業種差があります

### 導入企業
[ロゴ列：A社 B社 C社 D社 E社 F社（許諾済）]

### お客様の声
**A社（製造業 / 100名）人事部長 山田様（許諾済）**

「導入後3ヶ月で、採用工数が月150時間 → 月60時間（60%削減）。
朝、ダッシュボードを見るだけで状況把握できるようになりました。
上司から『楽になったね』と評価される日々です」

### 受賞・認証
- ISO 27001 認証取得
- Pマーク取得
- 「2025年注目のSaaS」（[業界誌名] 2025年12月号掲載）
```

### 型B: 同業種ストーリー型

```markdown
## あなたの業界で、すでにこんな成果が

### 製造業 / 50〜100名
- A社: 採用工数 60%削減
- B社: 内定辞退率 30%減

### IT・SaaS / 100〜300名
- C社: 応募数 1.5倍
- D社: 採用部門の満足度 90%

### 小売 / 30〜80名
- E社: 採用単価 40%減

※ 自社調べ・2025年実績。効果には個人差・業種差があります

[同業種事例の詳細を、無料資料に収録]
```

### 型C: ストーリー型お客様の声

```markdown
## A社（製造業 / 100名）の場合

### 導入前
人事部長は、毎週月曜日の朝が憂鬱でした。
スプレッドシートで応募者管理、抜け漏れが頻発。
「あの応募者、返信した？」と上司に聞かれるたびにヒヤヒヤ。

### 導入のきっかけ
営業からの強引な連絡がない、という資料の説明に安心。
専任担当による導入支援も決め手でした。

### 導入3ヶ月後
月150時間 → 月60時間（60%削減）。
朝、ダッシュボードを見るだけで状況把握。
上司から「楽になったね」と評価される日々。

「あの時、資料を請求して本当によかった」

— 人事部長 山田様（仮名・許諾済）
```

## 悪い例

### NG型1: 数値だけ並べる

```
- 1,200社導入
- 顧客満足度No.1
- 業界シェア1位
```

→ 数字は大きいが感情に届かない。出典なし。

### NG型2: テンプレお客様の声

```
「最高のサービスです！」
「すごく便利です！」
「使いやすいです！」
```

→ 信頼性ゼロ。

### NG型3: 出典なき推薦

```
「業界でも最高評価」
「専門家からも絶賛」
```

→ 法務リスク。

### NG型4: 顧客の許諾なき名前掲載

```
山田太郎様（実名）  株式会社○○（実社名）
```

→ 個人情報保護違反。

## 法的に問題のある表現と対応

### NG: No.1表現（根拠なし）
```
業界No.1の採用管理ツール
```

### OK: No.1表現（根拠付き）
```
[第三者調査会社] 調べ
「中小企業向け採用管理ツール」顧客満足度No.1
（2025年12月実績・対象者300名）
```

### NG: 効果断定
```
絶対に成果が出ます
100%の効果保証
```

### OK: 期待値表現
```
平均60%の業務工数削減（自社調べ・100社平均）
※ 効果には個人差・業種差があります
```

## 配置位置

| セクション | 配置すべき証拠 |
|---|---|
| FV | 信頼バッジ（実績数値・認証・受賞） |
| 戦略中段 | 認証セクションで詳述 |
| 解決策セクション後 | 数値実績 |
| 機能セクション後 | 比較表 / 受賞 |
| 導入事例セクション | お客様の声・ストーリー |
| 価格セクション直前 | 信頼補強（受賞・認証） |
| FAQ | 「効果はあるか？」への根拠提示 |
| フッター | 全権威要素一覧 |

## Validation 観点

- [ ] 最低3種類の証拠配置
- [ ] 全数値に出典（「自社調べ」でも明示）
- [ ] No.1表現に根拠（第三者調査・調査範囲・時期）
- [ ] 顧客許諾済み（実名・写真）
- [ ] 「個人差」明記
- [ ] 法務リスクなし
