# Photo Annotation Hero Plan - LP3 (運用代行)

## 案件
- 対象: LP3 (運用代行 / lp3-operation-outsourcing)
- 適用日: 2026-04-29

## 1. 写真選定
- **採用写真**: 採用担当者がデスクで業務している様子
- **業務文脈**: 採用業務の現場（書類・PC・電話対応）
- **採用理由**: ペルソナ（採用担当者）の業務シーン即時伝達

### 不採用候補
- 抽象オフィス風景: 業務文脈薄い
- ストックフォトの笑顔チーム: 実在感ゼロ
- 関係ない自然・都市風景: 商材無関係

## 2. オーバーレイ
- **種別**: ブランドカラー（紺）グラデーション
- **透過率 PC**: 60〜45%（写真が透ける）
- **透過率 SP**: 75〜60%（文字視認性優先）
- **視認性**: コントラスト 4.5:1 以上確認

```css
.lp3-hero-photo-overlay {
  background: linear-gradient(
    135deg,
    rgba(13, 71, 161, 0.65) 0%,
    rgba(13, 71, 161, 0.45) 100%
  );
}

@media (max-width: 640px) {
  .lp3-hero-photo-overlay {
    background: linear-gradient(
      135deg,
      rgba(13, 71, 161, 0.80) 0%,
      rgba(13, 71, 161, 0.65) 100%
    );
  }
}
```

## 3. メインコピー
- **メインキャッチ**: 「採用の運用、引き受けます」（13文字）
- **サブコピー**: 「中小企業の採用業務に専念できる体制」（19文字）

## 4. 注釈
- **配置**: 写真右上（業務量を指す）
- **内容**: 「業務時間 月150h → 月60h（自社調べ・100社平均）」
- **指線**: 細い白線（border 1px solid rgba(255,255,255,0.6)）

## 5. KPI バッジ
- **配置**: 写真左下（コーナーバッジ）
- **内容**: 「導入1,200社（自社調べ・2026年4月）」
- **内容2**: 「専任担当が3ヶ月伴走」

## 6. CTAボタン
- **文言**: 「3分で資料を受け取る」（11文字）
- **副CTA**: 「無料相談を予約」（8文字）

## 7. レスポンシブ
- **PC**: 写真背景全面（ブランドカラー オーバーレイ）+ テキスト中央寄せ
- **SP**: 写真縦長クロップ + テキスト中央集中
- **ブレークポイント**: 640px

## 8. CV貢献マップ

ペルソナ視線誘導順:
1. ターゲット明示「中小企業の採用担当者へ」（自分ごと化）
2. メインキャッチ「採用の運用、引き受けます」（メリット）
3. KPIバッジ「導入1,200社」（信頼）
4. 注釈「月150h → 月60h」（期待）
5. CTAボタン（行動）

## 採用理由ログ

### 写真採用理由
- 運用代行は「人が業務を引き継ぐ」感情訴求が重要
- 業務シーン写真でペルソナ即時共感
- 競合SaaSの画面UI訴求から差別化

### 注釈採用理由
- 単なる写真背景は装飾的
- 注釈で「具体的な改善数値」を可視化
- 写真の特定箇所（PC作業量）を指して意味付け

### KPI バッジ採用理由
- 数字で実在感
- 「専任担当」で運用代行の特長訴求
- コーナー配置で視線誘導
