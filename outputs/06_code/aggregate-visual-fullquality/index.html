差別化の方針

4本とも配色だけでなく、レイアウト・タイポ・ビジュアル合成・モーション・訴求順序を変えています。

  • LP1: 帳票感・大数値・セリフ強調
  • LP2: 浮遊カード・成長感・ファネル
  • LP3: 横タイムライン・チェック演出
  • LP4: エディトリアル・PDCA・チャート

画像の扱い

「写真を差し込む」ではなく、SVG で構成された訴求装置として設計。グラデ・編み・KPIカード・図解・UIモックを統合しています。

  • すべての主要ビジュアルを外部 SVG 化
  • 画像 + 短コピー + 数値訴求を統合
  • base64 埋め込み軽量逃げは禁止

実装方針

共通基盤と LP 個別を完全分離。reset / common / LP 別 CSS、common / LP 別 JS、SVG 個別ファイル化で保守性を確保しています。

  • CSS 6 ファイル (reset / common / lp1-4)
  • JS 5 ファイル (common / lp1-4)
  • SVG ビジュアル外部アセット化
