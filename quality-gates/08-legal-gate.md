# Gate 8: Legal Gate

## 目的

法令違反・誇大表現・優良誤認のリスクを排除する。

## 担当エージェント

- `lp-legal-expression-checker`
- `lp-security-checker`

## 合格条件

- [ ] No.1表現の根拠あり
- [ ] 実績表現の根拠あり
- [ ] Before/After 表現が誇大でない
- [ ] 断定表現に注釈
- [ ] 誤認リスクなし
- [ ] 景表法・薬機法・特商法 準拠
- [ ] 個人情報取扱が法令準拠
- [ ] プライバシーポリシーリンクあり

## 不合格条件

- 出典なき数値
- 根拠なきNo.1表現
- 効果効能の業法違反
- 個人情報の同意取得なし

## 差し戻し条件

- 上記不合格条件
- 業界該当法令の確認なし

```markdown

```
