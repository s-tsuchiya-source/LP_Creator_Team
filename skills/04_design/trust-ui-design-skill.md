# Trust UI Design Skill

## Purpose

LPの信頼性をデザインで担保する（信頼を毀損しないUI / 信頼を高めるUI）。

## When to Use

- BtoB / 高単価 / 採用 / 医療 等の信頼重視LP

## Required Inputs

- ペルソナ（信頼基準）
- ブランドガイドライン
- 業種playbook

## Step-by-Step Procedure

1. 信頼を毀損する要素を排除
   - 過度なアニメーション
   - フリー素材の人物写真の多用
   - 派手すぎる配色
   - タイポミス
2. 信頼を高める要素を配置
   - 企業ロゴ列
   - 認証バッジ
   - 顧客の顔写真（許諾済み）
   - 数値実績（出典付き）
3. 余白を十分に取る
4. 落ち着いた配色を基調に
5. プロ品質の写真を使用
6. `outputs/05_design/trust-ui.md` に保存

## Output Format

```markdown
## Trust UI Direction

### 信頼を高める要素
- 企業ロゴ列の位置・サイズ
- 認証バッジ（ISO / Pマーク）の配置
- 顧客顔写真の許諾状況

### 信頼を毀損する要素の排除
- アニメーション軽量化
- フリー素材使用範囲

### 配色
- メインカラー（落ち着き）:
- アクセントカラー（CTA用）:
```

## Quality Criteria

- 信頼毀損要素を排除
- 信頼向上要素を配置
- 業界文脈に合うトーン

## Common Failure Patterns

- 派手な配色で信頼毀損
- フリー素材で安っぽく
- 認証バッジを目立たせず

## Self Check

- [ ] 毀損要素排除済み
- [ ] 向上要素配置済み
