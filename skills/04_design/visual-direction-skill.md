# Visual Direction Skill

## Purpose

LP全体のビジュアル方針（トーン / カラー / タイポ / 写真 / イラスト）を決定する。

## When to Use

- デザインフェーズ序盤

## Required Inputs

- ブランドガイドライン（あれば）
- ペルソナ
- 訴求軸
- 業種playbook

## Step-by-Step Procedure

1. トーンを形容詞3つで定義
2. カラーパレット決定（メイン / アクセント / ベース / テキスト）
3. コントラスト比を確認（4.5:1以上）
4. タイポグラフィ選定（見出し / 本文 / 日本語 / 英数字）
5. 写真テイスト（自然光 / モノクロ / カラフル）
6. イラスト・アイコン方針
7. NG表現リスト
8. `outputs/05_design/visual-direction.md` に保存

## Output Format

`templates/lp-design-brief-template.md` に準拠。

## Quality Criteria

- ブランドガイドライン準拠
- コントラスト比4.5:1以上
- タイポ階層明確
- ペルソナ感性に合う

## Common Failure Patterns

- ガイドライン違反
- コントラスト不足
- トーン定義が曖昧
- ターゲット感性ズレ

## Self Check

- [ ] ガイドライン準拠
- [ ] コントラスト合格
- [ ] トーン3形容詞で定義
