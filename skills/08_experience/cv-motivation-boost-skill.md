# CV Motivation Boost Skill

## Purpose

CV直前の行動意欲を高め、最終CTAでの離脱を防ぐ。

## When to Use

- 最終CTAセクション設計時
- 限定性・緊急性の検討時
- CTAボタン文言の最適化時

## Required Inputs

- ペルソナの動機マップ
- オファー設計
- 業種 playbook

## Step-by-Step Procedure

### 行動意欲を高める7要素

```
1. 最終CTA前の心理状態を理解
2. 今行動する理由
3. 資料を見るメリット
4. 問い合わせするメリット
5. 無料相談するメリット
6. CTA文言の改善
7. CV後の期待感
```

### CV直前の心理状態

```
✓ 「資料くらいなら見てもいい」
✓ 「営業されすぎないか不安」
✓ 「フォーム面倒くさそう」
✓ 「申込後の流れが見えない」
```

→ これら全てに対応する

### 手順

1. 最終CTAセクションの前に「行動意欲」要素を配置
2. ハードル低いCTA（資料請求 or 無料相談）を主CTA
3. ハードル高いCTA（契約 or 購入）は副CTA
4. 限定性は実態に即して
5. CV後の期待感を提示
6. CTAボタン文言を「動詞 + ベネフィット」に

## Output Format

### 最終CTAセクション

```markdown
## [サービス名] で、採用業務を変えませんか？

✓ 採用工数を最大50%削減
✓ 応募者対応の質を向上
✓ 経営層への報告が自動化

導入企業1,200社が選ぶ採用管理ツール

[今すぐ無料で資料請求]   [無料デモを予約]

★ 14日間無料トライアル / クレジットカード不要
★ いつでも解約可能
★ 専任担当による導入支援

### 資料請求のメリット

- 同業種事例 5社
- ROI試算シート
- 機能比較表（競合A・B社 vs 自社）
- セキュリティチェック資料

### 個別相談のメリット

- 貴社の課題に合わせた個別アドバイス
- 30分・オンライン
- 強引な営業は一切なし
```

### CTAボタン文言の例

| ハードル低 | ハードル中 | ハードル高 |
|---|---|---|
| 「無料で資料を受け取る」 | 「無料デモを予約する」 | 「今すぐ申し込む」 |
| 「3分で資料DL」 | 「個別相談（無料）」 | 「契約手続きへ」 |
| 「メリット診断する」 | 「無料トライアル開始」 | - |

### 「今行動する理由」の作り方

```markdown
### 期間限定の特典

✓ 14日間無料トライアル（通常7日間）
✓ 初期設定支援（通常別途料金）
✓ データ移行代行（通常別途料金）

※ 2026年X月X日まで（実態に即した限定）
```

### CV後の期待感

```markdown
資料請求後、こんな価値があります：

📊 **すぐに**: 資料で機能・事例・価格が確認できる
💬 **任意で**: 個別相談で貴社課題を整理
🎯 **無料で**: 14日間トライアルで実環境テスト
🚀 **そして**: 業務改善の道筋が見える
```

## Quality Criteria

- 最終CTAセクションに7要素全て配置
- ハードル低 / 中 / 高 の3段階CTA
- 限定性は実態に即す
- CV後の期待感を提示
- CTA文言が「動詞 + ベネフィット」

## Common Failure Patterns

- CTA1つだけ（ハードル選択肢なし）
- 「お問い合わせ」だけ
- 嘘の限定性（常時セール）
- CV後の期待感なし
- 「今行動する理由」が弱い

## Self Check

- [ ] 7要素全て配置

- [ ] 限定性は実態に即す
- [ ] CV後の期待感あり
- [ ] CTA文言が具体的
