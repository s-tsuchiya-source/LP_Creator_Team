# Emotional Objection Handling Skill

## Purpose

論理的な反論だけでなく、感情的な不安を処理し、CV直前の心理ハードルを下げる。

## When to Use

- FAQ セクション作成時
- 価格セクション作成時
- 最終CTA直前のコピー作成時

## Required Inputs

- ペルソナ
- 営業ヒアリング情報（あれば）
- インサイト整理書（反論リスト）

## Step-by-Step Procedure

### 感情的な不安7カテゴリ

```
1. 失敗したくない
2. 怒られたくない
3. 比較が面倒
4. 社内説明が不安
5. 時間がない
6. 本当に効果があるか不安
7. 問い合わせ後に営業されすぎるのが不安
```

### 各不安に対応する手順

1. ペルソナごとに7カテゴリの感情不安を確認
2. 各不安に対する「共感 → 論拠 → 証拠」コピーを作成
3. FAQ セクションに配置
4. 最終CTA直前に主要不安解消コピーを配置
5. 法的に問題のある保証は避ける

## Output Format

### 失敗したくない不安への対応

```markdown
**Q. 導入して失敗したらどうすればいいですか？**

A. 多くの方が同じ不安をお持ちです。

そのため、[サービス名] では：
- 月単位の解約OK（最低利用期間なし）
- 14日間無料トライアル中の解約自由
- データはいつでもCSVエクスポート可能

「契約後も、自由に判断してください」が私たちの方針です。
実際、トライアル後の継続率は85%（自社調べ・2025年実績）です。
```

### 怒られたくない不安への対応

```markdown
**Q. 上司に怒られないか心配です**

A. 安心してください。

[サービス名] には、上司向けの説明資料が無料で含まれています：
- ROI試算シート（あなたの会社規模で計算済み）
- 同業種の導入事例集
- セキュリティチェックリスト
- 経営層向け1pサマリー

社内会議でそのまま使える素材を、事前にご用意します。
```

### 営業されすぎ不安への対応

```markdown
資料請求後の対応について：

✓ 営業からの強引な連絡はありません
✓ ご希望の方のみ、個別相談を実施
✓ メルマガはご希望に応じて
✓ 解除はいつでも可能

「資料を見るだけ」でもOKです。
```

## Quality Criteria

- 7カテゴリ全てに対応コピーがある
- 「共感 → 論拠 → 証拠」構造
- 法的に問題のある保証なし
- ペルソナの言葉で書かれている

## Common Failure Patterns

- 「ご安心ください」だけ（具体性なし）
- 「絶対」「100%」の保証
- 不安への直接対応を避ける
- 不利情報を隠蔽
