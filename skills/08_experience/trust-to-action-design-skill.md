# Trust to Action Design Skill

## Purpose

信頼を行動に変える設計を行う。信頼だけでは CV しない、その「最後の一押し」を設計する。

## When to Use

- CTA直前のセクション設計時
- フォーム周辺のコピー作成時
- 最終CTAセクション設計時

## Required Inputs

- 完成LPの信頼セクション（実績・事例・認証）
- フォーム仕様
- CV後の流れ

## Step-by-Step Procedure

### 信頼 → 行動変換の6要素

```
1. 信頼だけではCVしない（追加要素必要）
2. 信頼のあとに必要なのは具体的な次アクション
3. CTA直前に置くべき情報
4. CV後の流れの提示
5. フォーム入力の安心感
6. 資料請求・問い合わせの心理的負担を下げる方法
```

### 手順

1. 信頼セクション（実績・事例・認証）を確認
2. その後すぐに「次アクション」を提示
3. CV後の流れを最終CTA直前に明示
4. フォーム周辺に安心情報を配置
5. CTAボタンの文言を「ハードル低い行動」に

## Output Format

### CTA直前セクション

```markdown
## まずは資料を見てみませんか？

導入企業1,200社が選ぶ採用管理ツール、
3分で送られる資料には、以下が含まれます：

✓ 機能一覧と業務適用例
✓ 同業種の導入事例 5社
✓ 価格・プラン詳細
✓ ROI試算シート
✓ セキュリティ情報

資料請求後の流れ：
1. メールで資料が届きます（即時）
2. ご希望の方のみ、個別相談を実施
3. 営業からの強引な連絡はありません
4. メルマガ登録は任意です

[3分で資料を受け取る]   [個別相談を予約]

★ クレジットカード不要
★ 個人情報の利用範囲はプライバシーポリシーをご確認ください
```

### サンクスページ コピー

```markdown
## 資料請求ありがとうございます

ご記入いただいたメールアドレスに、
資料を送信いたしました。

### 次の流れ

1. **すぐに**: 資料が届きます（迷惑メールフォルダもご確認ください）
2. **3営業日以内**: 担当者からご挨拶メールをお送りします
3. **ご希望時**: 個別相談・デモのご予約が可能です

### よくあるご質問

- Q. 資料が届かない場合は？
- Q. 営業電話はありますか？
- Q. デモを予約したいのですが？

### 関連資料

- [競合比較表 PDF]
- [ROI試算シート]
```

## Quality Criteria

- 信頼セクションの直後に次アクションを配置
- CV後の流れが明示
- フォーム周辺に安心情報
- CTAボタン文言が「ハードル低い行動」
- サンクスページが安心感を提供

## Common Failure Patterns

- 信頼セクションで終わる（次アクションなし）
- CV後の流れが不明
- フォーム周辺の安心情報なし
- CTA「お問い合わせ」だけ
- サンクスページが汎用的すぎる
