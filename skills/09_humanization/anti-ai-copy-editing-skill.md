# Anti-AI Copy Editing Skill

## Purpose

AIっぽいコピーを人間らしく編集する。

## When to Use

- コピー完成後の最終チェック
- AIっぽさが検出されたとき
- 競合置き換えテストで成立してしまうコピーがあるとき

## Required Inputs

- 編集対象コピー
- ペルソナ
- 商材情報・出典付き数値

## Step-by-Step Procedure

### 7段階変換手順

```
1. 抽象語を抽出
2. 競合に置き換えても成立する表現を削除
3. 読み手の状況に寄せる
4. 機能説明を利用シーンに変える
5. メリットを業務・心理・評価に分解
6. 売り込み臭を下げる
7. CTA直前に安心材料を追加
```

### 詳細手順

#### Step 1: 抽象語を抽出

LPコピーから以下の語を全てリストアップ：

```
最適化 / 革新的 / 寄り添う / 課題解決 / シームレス
業界をリード / お客様第一 / 唯一無二 / 業界初
最高品質 / 革命的 / パラダイムシフト
```

#### Step 2: 競合置き換えテスト

各文を競合社名に置き換えて、成立するか確認。
成立する文は AIっぽい → 削除 or 具体化。

#### Step 3: 読み手の状況に寄せる

抽象的な訴求を、ペルソナの業務シーンに変換：

```
Before: 業務効率を上げる
After: 月150時間の手作業を、月60時間に減らす
```

#### Step 4: 機能 → 利用シーン変換

```
Before: API連携機能
After: 求人媒体から応募者情報が自動で取り込まれる
       毎週月曜の手作業のコピペは、もう不要
```

#### Step 5: ベネフィットを4軸で分解

```
業務メリット: 月150時間 → 月60時間
心理メリット: 「これで失敗しないと思える」
評価メリット: 上司から「楽になったね」
未来メリット: 来期、戦略採用に時間を使える
```

#### Step 6: 売り込み臭を下げる

```
Before: 業界最高のサービスをぜひお試しください！
After: 資料を見るだけ、でもOKです。
       営業からの強引な連絡はありません。
```

#### Step 7: CTA直前に安心材料追加

```
[CTA: 3分で資料を受け取る]

★ クレジットカード不要
★ いつでも解約可能
★ 営業からの強引な連絡なし
```

## Output Format

### 編集ログテンプレート

```markdown
# Anti-AI Copy Editing Log

## 抽象語リストアップ
| 元の表現 | 修正後 |
|---|---|
| 最適化 | 月150時間 → 月60時間（60%削減） |
| ... | ... |

## 競合置き換え不成立化
| 元の文 | 修正後の文 |
|---|---|
| 私たちは業界をリードします | [サービス名] は、月150時間の採用業務を月60時間に削減します |

## 機能 → 利用シーン変換
| 機能名 | 利用シーン化 |
|---|---|
| API連携 | 月曜朝のコピペ作業がゼロに |

## 売り込み臭の修正
| Before | After |

## CTA直前の安心材料追加
- 追加した安心要素一覧
```

## Quality Criteria

- 抽象語の連続使用（3つ以上）なし
- 競合置き換えテスト全文不成立
- 業務シーン描写あり
- ベネフィット4軸分解
- 売り込み臭低減
- CTA直前の安心要素

## Common Failure Patterns

- 抽象語を別の抽象語に置き換える
- 数字に出典を付け忘れる
- 売り込み臭を消しすぎて訴求が弱くなる
- CTA を弱めすぎる
