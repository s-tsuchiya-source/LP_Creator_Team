# Image Direction Humanization Skill

## Purpose

画像・イラストの指示を、AIっぽさを排除して人間らしく具体化する。

## When to Use

- 画像生成 / ストックフォト選定の前
- AI生成画像が不自然なとき
- 画像が商材と関係ない印象のとき

## Required Inputs

- 商材情報
- ペルソナ
- 業種 playbook
- 配置セクション（FV / 解決策 / 実績 等）

## Step-by-Step Procedure

### 5つの指示要素

```
1. 実際の利用シーンを定義
2. ターゲットの状況を描く
3. 装飾目的の画像を排除
4. AI生成感のある不自然なディテールを避ける
5. 図解・実画面・現場写真の優先順位を決める
```

### 詳細手順

#### Step 1: 利用シーン定義

各セクションで、何を伝えたいかを明確化：

```
FV: ターゲットが実際にプロダクトを使っているシーン
解決策: 機能・業務フローの図解
実績: 導入企業のロゴ・お客様の写真
事例: ストーリー型の写真コラージュ
```

#### Step 2: ターゲットの状況描写

ペルソナの業務文脈に合わせる：

```
BtoB SaaS（採用管理）:
✅ 人事担当者がデスクで管理画面を見ているシーン
✅ ダッシュボードのスクリーンショット
✅ 経営層への報告風景
❌ 抽象的な業務イメージ
❌ ストックフォトのチームミーティング

採用LP（エンジニア）:
✅ 実際のオフィス・社員写真
✅ コードエディタ画面
✅ ペアプロ風景
✅ 1on1 の様子
❌ AI生成のキャラクター
❌ ストックフォトの開発者
```

#### Step 3: 装飾目的の画像排除

「あってもなくてもいい」画像は削除：

- 抽象未来都市
- 光の粒子
- 抽象的な背景画像
- 関係ない風景

#### Step 4: AI生成感の不自然さ排除

AI生成画像を使う場合のチェック：

```
NG（AI生成感）:
- 不自然な手・指（6本指、変な角度）
- 不自然な顔（左右非対称）
- 不自然な光沢・質感
- 背景の溶けた人物
- 文字が読めない看板

OK（自然）:
- 商材文脈に合う
- 装飾ではなく理解促進
- 不自然なディテールなし
- 著作権・利用規約準拠
```

#### Step 5: 優先順位

```
1位: 実画面・実プロダクト（最高優先）
   └ SaaS 管理画面、EC 商品写真

2位: 実際の利用シーン
   └ 業務風景、社員写真

3位: 図解・インフォグラフィック
   └ 業務フロー、Before/After

4位: アイコン
   └ Material / Lucide

5位: ストックフォト（最低限）
   └ 補助的に、文脈に合う場合のみ
```

## Output Format

### 画像指示ログ

```markdown
# Image Direction Humanization Log

## セクション別画像方針

### FV
- 役割: 商材の利用シーンを伝える
- ビジュアル: プロダクト管理画面（モックアップ）
- AI生成画像使用: なし / 限定的

### 課題提起
- 役割: 担当者の業務シーン共感
- ビジュアル: 業務シーンの写真
- 注意: ストックフォト避ける、リアル感優先

### 解決策
- 役割: 機能を視覚化
- ビジュアル: スクリーンショット + アイコン
- 注意: 装飾目的の画像なし

### 実績
- 役割: 信頼性訴求
- ビジュアル: 導入企業ロゴ列、お客様写真（許諾済）
- 注意: フリー素材の人物NG

### 最終CTA
- 役割: 安心感
- ビジュアル: なし or バッジ風アイコン
- 注意: 装飾過多にしない

## AI生成画像チェック
- [ ] 不自然な手・指なし
- [ ] 不自然な顔なし
- [ ] 商材文脈に合う
- [ ] 著作権・利用規約準拠

## 画像優先順位
1. 実画面・実プロダクト: [配置箇所]
2. 実際の利用シーン: [配置箇所]
3. 図解: [配置箇所]
4. アイコン: [配置箇所]
5. ストックフォト: [配置箇所と理由]
```

## Quality Criteria

- 全画像に役割が明示
- 商材文脈に合う
- AI生成感の不自然さなし
- 装飾目的の画像なし
- 優先順位に沿った選定

## Common Failure Patterns

- 画像を装飾だけのために配置
- 商材文脈を無視したストックフォト
- AI生成画像の不自然さを見逃す
- 全部実写真で重くなる

## Self Check

- [ ] 各画像の役割明示
- [ ] 商材文脈整合
- [ ] AI生成感なし
- [ ] 優先順位通り
