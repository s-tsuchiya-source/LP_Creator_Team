## 総合判定

- [ ] A: そのまま本番候補
- [ ] B: 軽微修正で可
- [ ] C: 中修正が必要
- [ ] D: 大幅修正が必要
- [ ] F: 不採用

## 1. FV

- [ ] 3秒以内に価値が伝わる
- [ ] コピーが読める
- [ ] CTAが見つかる
- [ ] 画像が単なる飾りになっていない

## 2. 課題セクション

- [ ] 流入者の悩みが視覚化されている
- [ ] 課題ラベルや図解が理解を助けている
- [ ] 不安を煽りすぎていない

## 3. 解決セクション

- [ ] どう解決するかが図解されている
- [ ] UIモックや数値が意味を持っている
- [ ] 文章だけに依存していない

## 4. CTA前

- [ ] 相談しやすさが伝わる
- [ ] CTA直前の不安が下がっている
- [ ] 余計な装飾でCTAを邪魔していない

## 5. スマホ

- [ ] 画像の主題が切れていない
- [ ] テキストが読める
- [ ] CTAが押しやすい
- [ ] 横スクロールがない

## 6. 法務・事実性

- [ ] 根拠不明の数値がない
- [ ] 実在ロゴを無断使用していない
- [ ] 誇張表現がない
