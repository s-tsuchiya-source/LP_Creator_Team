# Emotional CVR Check

LPの感情設計とCV直結性を検証する専用チェックリスト。

## 必須チェック8項目

### 1. メリットが明確か
- [ ] 機能ではなくベネフィットで訴求
- [ ] 「あなたの業務がこう変わる」が具体的
- [ ] 数値根拠（出典付き）あり
- [ ] 短期 / 中期 / 長期 メリットを網羅

### 2. 自分ごと化できるか
- [ ] FV にターゲット明示
- [ ] 「こんな悩みありませんか」セクションあり
- [ ] 言語化されていない悩みを 3〜5個提示
- [ ] ペルソナの言葉で書かれている

### 3. 期待感があるか
- [ ] 数値根拠（4種: 数値・事例・プロセス・第三者）配置
- [ ] 導入後の時系列（1ヶ月 / 3ヶ月 / 6ヶ月）
- [ ] 「個人差・業種差」明記
- [ ] 誇大表現排除

### 4. 導入後の未来が想像できるか
- [ ] 4つの未来（現状 / 放置 / 導入後 / 成果後）
- [ ] 一人称表現
- [ ] 五感に訴える描写
- [ ] 周囲の反応も含む

### 5. 不安が解消されているか
- [ ] 6カテゴリ不安全てに対応
  - 価格不安
  - 導入不安
  - 効果不安
  - 運用負荷不安
  - 社内説明不安
  - 失敗不安
- [ ] FAQ 5〜10件
- [ ] 「契約後も自由」設計

### 6. CTAに進む理由があるか
- [ ] ハードル段階別CTA（最低2個）
- [ ] CTA文言が「動詞 + ベネフィット」
- [ ] 「お問い合わせ」だけになっていない
- [ ] 「今動く理由」明示
- [ ] 限定性は実態に即す

### 7. CV後の流れが分かるか
- [ ] 最終CTA直前に「CV後の流れ」
- [ ] サンクスページ設計
- [ ] 営業電話の有無を明示
- [ ] メルマガ登録の任意性
- [ ] 「資料を見るだけ」OK

### 8. 誇大表現ではないか
- [ ] 「絶対」「100%」「必ず」なし
- [ ] No.1 表現に根拠
- [ ] Before / After に注釈
- [ ] 法務リスククリア

## 9段階感情ジャーニー検証

LP全体を通読し、以下の感情変化が起きるか：

```
[1. 流入] → 興味
[2. 自分ごと化] → 共感
[3. 課題共感] → 緊張
[4. 期待] → わくわく
[5. 信頼] → 安心
[6. 不安解消] → 確信
[7. 行動意欲] → 推進力
[8. CV直前] → 決断
[9. CV後] → 安心
```

各段階で対応LP要素が機能しているか確認。

## 採点

8項目のうち何項目満たすかで採点：

- 8/8: 100点（最高品質）
- 7/8: 90点（納品可能）
- 6/8: 80点（軽微修正）
- 5/8 以下: 主要改善必須

`scoring/emotional-cvr-scorecard.md` での採点結果と整合させる。
