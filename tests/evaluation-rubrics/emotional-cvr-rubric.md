# Rubric: Emotional CVR

## 評価基準

| 軸 | 5点 | 3点 | 1点 |
|---|---|---|---|
| 自分ごと化 | FV 明示 + 悩みリスト + 業務シーン | 一部のみ | 抽象訴求のみ |
| メリット明確 | 4軸分解 + 数値根拠 | 数値あり、抽象 | 機能羅列 |
| 期待感 | 4種根拠 + 時系列 + 「個人差」明記 | 一部根拠 | 誇大or根拠なし |
| 不安解消 | 6カテゴリ網羅 + 「契約後自由」 | 一部カテゴリ | FAQ なし |
| CV直前 | ハードル段階別 + 「今動く理由」 + CV後流れ | 一部のみ | 「お問い合わせ」だけ |

## 採点

5点 × 5軸 = 25点満点

- 23+: 優（合格）
- 19〜22: 良（軽微修正）
- 18以下: 差し戻し

## 関連 scorecard 必須スコア

```
emotional-cvr-scorecard: 85+
benefit-clarity-scorecard: 85+
expectation-scorecard: 85+
self-relevance-scorecard: 85+
```

すべて達成で合格。

## チェック項目

### 自分ごと化
- [ ] FV ターゲット明示
- [ ] 「こんな悩みありませんか」3〜5問
- [ ] 業務シーン描写（「月曜の朝」等）

### メリット明確
- [ ] 機能 → ベネフィット翻訳済み
- [ ] 業務 / 心理 / 評価 / 未来 の4軸分解
- [ ] 数値根拠（出典付き）

### 期待感
- [ ] 数値根拠
- [ ] 事例根拠
- [ ] プロセス根拠
- [ ] 第三者根拠
- [ ] 導入後時系列（1ヶ月 / 3ヶ月 / 6ヶ月）
- [ ] 「個人差」明記

### 不安解消
- [ ] 価格不安対応
- [ ] 導入不安対応
- [ ] 効果不安対応
- [ ] 運用負荷不安対応
- [ ] 社内説明不安対応
- [ ] 失敗不安対応

### CV直前
- [ ] ハードル段階別CTA（最低2個）
- [ ] 「今動く理由」（実態に即す）
- [ ] CV後の流れ予告
- [ ] 営業電話の有無を明示
