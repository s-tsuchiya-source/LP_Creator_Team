# Rubric: Human Quality

## 評価基準

LP出力の「人間らしさ」と「AIっぽさの排除度」を評価。

| 軸 | 5点 | 3点 | 1点 |
|---|---|---|---|
| 競合置き換えテスト | 90%以上不成立 | 70〜89% | 70%未満 |
| 業務シーン描写 | 全セクション | 一部 | なし |
| 担当者の本音 | 自然に織り込み | 一部 | なし |
| 数字の出典 | 全数値に出典 | 一部 | 出典なし |
| 商材文脈ビジュアル | 全画像整合 | 一部装飾 | 抽象 hero |
| 売り込み臭の低さ | 「資料を見るだけOK」 | 一般的 | 押し売り |
| CTA行動理由 | 明確 + 安心要素 | 文言適切 | 「お問い合わせ」だけ |

## 採点

5点 × 7軸 = 35点満点

- 32+: 優（合格）
- 26〜31: 良（軽微修正）
- 25以下: 差し戻し

## 関連 scorecard 必須スコア

```
human-naturalness-scorecard: 85+
anti-ai-smell-scorecard: 90+
```

## Critical Fail（点数に関係なく差し戻し）

❌ どの会社にも当てはまるFV
❌ 抽象語連続使用（5語以上）
❌ AI生成画像の不自然さ多数
❌ 実在感のない事例
❌ 「絶対」「100%」誇大表現
❌ 出典なき断定数値

## チェック項目

### 競合置き換えテスト
- [ ] LPコピー全体を競合社名置き換え
- [ ] 90%以上の文が不成立 = 自社固有

### 業務シーン
- [ ] 「月曜の朝」等の特定瞬間
- [ ] ペルソナの業務文脈
- [ ] 業界一般論なし

### 担当者の本音
- [ ] 「失敗したくない」「上司に怒られたくない」等
- [ ] 自然に織り込まれている

### 数字
- [ ] 全数値に出典
- [ ] 「※個人差」明記

### ビジュアル
- [ ] 商材文脈に合う
- [ ] 抽象 hero なし
- [ ] AI生成感の不自然さなし

### 売り込み臭
- [ ] 「資料を見るだけOK」
- [ ] 「営業電話なし」明示
- [ ] 「お断り歓迎」明示

### CTA
- [ ] 「動詞 + ベネフィット」
- [ ] 安心要素（解約・返金・営業電話）
- [ ] CV後の流れ
