# Expected: AIっぽさ排除

## 期待される LP 出力

### 1. 危険語の連続使用なし

以下の語が連続3つ以上使われていない：

```
最適化 / 革新的 / 寄り添う / 課題解決 / シームレス
業界をリード / お客様第一 / 唯一無二 / 業界初 / 最高品質
```

### 2. 競合置き換えテスト不成立

LPコピーを競合社名に置き換えて、90%以上の文が成立しない。

### 3. 業務シーン描写あり

```
✅ 「月曜の朝、応募者リストを開くのが憂鬱」
✅ 「上司から『あの応募者、返信した？』と聞かれてヒヤヒヤ」
✅ 「夜眠れない時がある」
```

### 4. 担当者の本音入り

```
✅ 「失敗したくない」
✅ 「上司に怒られたくない」
✅ 「胸を張って報告したい」
```

### 5. 全数値に出典

```
✅ 「導入企業1,200社（自社調べ・2026年4月時点）」
✅ 「平均60%削減（自社調べ・100社平均）」
✅ 「[第三者調査] 調べ・対象者300名・2025年12月」
```

### 6. CTA に行動理由

```
✅ 「3分で資料を受け取る」
✅ 「14日間無料で試す（クレカ不要）」
✅ 「カジュアル面談を予約する」
```

### 7. 商材文脈に合うビジュアル

```
✅ BtoB SaaS → プロダクト管理画面
✅ 採用LP → 実在社員写真
✅ EC → 実商品写真
```

### 8. CV後の流れ明示

```
✅ 営業電話の有無
✅ メルマガ任意性
✅ 申込後3営業日以内の対応
```

## NG パターン

❌ どの会社にも当てはまるFV
❌ 抽象語だらけのコピー
❌ 「絶対」「100%」誇大表現
❌ 出典なき断定数値
❌ FAQ「ご安心ください」だけ
❌ CTA「お問い合わせ」だけ
❌ AI生成画像の不自然さ
❌ 装飾だけのストックフォト

## 必須採点

```
- human-naturalness-scorecard: 85+
- anti-ai-smell-scorecard: 90+
- 競合置き換えテスト: 90%以上不成立
```
