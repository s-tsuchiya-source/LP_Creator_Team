# Expected: Emotional CVR Layer 反映

## 期待される LP 出力

### 9段階感情ジャーニー反映

| 段階 | 期待感情 | 必須セクション |
|---|---|---|
| 1. 流入 | 興味 | FV（信頼バッジあり） |
| 2. 自分ごと化 | 共感 | FV ターゲット明示 + 課題チェックリスト |
| 3. 課題共感 | 緊張 | 業務シーン描写 + 放置リスク |
| 4. 期待 | わくわく | 解決策 + 未来像（時系列） |
| 5. 信頼 | 安心 | 実績（出典付き） + 認証 |
| 6. 不安解消 | 確信 | FAQ 6カテゴリ網羅 |
| 7. 行動意欲 | 推進力 | 最終CTA前のオファー再訴求 |
| 8. CV直前 | 決断 | 最終CTA + ハードル下げ |
| 9. CV後 | 安心 | サンクスページ設計 |

### 必須採点

```
- emotional-cvr-scorecard: 85+
- benefit-clarity-scorecard: 85+
- expectation-scorecard: 85+
- self-relevance-scorecard: 85+
```

### 「自分ごと化」期待要素

- FV「中小企業の人事担当者へ」明示
- 「こんな悩みありませんか」チェックリスト 3〜5個
- 業務シーン描写（「月曜の朝」「上司から聞かれる」等）

### 「期待感」期待要素

- 数値根拠（出典付き）
- 同業種事例
- 導入後の時系列描写
- 「個人差・業種差」明記

### 「不安解消」期待要素

- 6カテゴリ FAQ
- 「契約後も自由」設計
- 「営業されすぎない」明示

### 「CV直前」期待要素

- ハードル段階別CTA
- 「今動く理由」（実態に即す）
- CV後の流れ予告

## NG パターン

❌ 機能羅列のみ（期待感ゼロ）
❌ 「自分ごと化」要素皆無
❌ 不安解消セクション欠落
❌ CV後の流れ完全不明
❌ 「絶対」「100%」誇大表現

## 採点基準

各 Emotional CVR scorecard を採点し、4つすべて 85+ で合格。
