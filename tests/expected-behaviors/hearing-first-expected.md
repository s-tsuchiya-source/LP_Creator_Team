# Expected: ヒアリングから始まる

## 期待される lp-director の初回応答

### パターンA: ヒアリング実施

```
ご依頼ありがとうございます。

最高品質のLPを制作するため、まず以下12問以内のヒアリングをさせてください。
すべて回答必須ではなく、不明点は「仮置きでOK」と明示いただければ進められます。

1. 商材・サービス名は何ですか？
2. LPの目的は資料請求 / デモ予約 / 無料トライアル のいずれですか？
3. 主要ターゲットは誰ですか？（業種・規模・役職）
4. ターゲットが抱えている悩みは何ですか？
5. サービスの主な提供価値は何ですか？
6. 競合・代替手段は何ですか？
7. 強み・差別化要素は何ですか？
8. 実績・事例・数字はありますか？
9. 希望するトーンやデザインイメージは？
10. 法務リスク該当業種ですか？（医療 / 美容 / 金融等）
11. 公開期日はありますか？
12. 出力形式は戦略のみ / コピー込み / 実装込み のどれですか？

特に、CV / ターゲット / 商材内容 / 法務リスク は確認必須です。
それ以外は仮置きで進めることが可能です。
```

### パターンB: 仮置き提示 + 確認

```
情報が限定的なため、以下を仮置きで進めます。

【仮置き項目】
- ターゲット: 中小企業の人事担当者（30〜45歳）
- CV: 資料請求
- 訴求軸: 採用工数削減
- 業種 playbook: btob-saas-playbook.md
- 出力形式: 戦略 + コピー（実装は別途）

【要確認】
- 商材の具体機能
- 既存実績の数値（仮置きなら「業界平均」を使用）
- 法務リスクの有無

進めてよろしいでしょうか？
```

## NG パターン

❌ いきなりLPを生成し始める
❌ 12問を超えるヒアリング
❌ 仮置きを明示せず勝手に決め打ち
❌ 重要不明点を仮置きで進める

## 採点基準

- ヒアリング or 仮置き提示: PASS
- 12問以内: 1点減点なし / 13問以上: -10
- 重要不明点を確認している: PASS / 確認なし: -20
- 仮置き明示: 1点減点なし / 明示なし: -10

合計 80+ で合格。
