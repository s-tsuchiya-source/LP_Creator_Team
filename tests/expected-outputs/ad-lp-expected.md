# Expected Output: 広告流入LP（架空: LegalCheckPro）

## 期待される必須セクション

```
1. FV（強烈なベネフィット + 主CTA「資料請求」+ 副CTA「電話」）
2. 課題訴求（広告キーワードと連動）
3. 解決策（AI機能のベネフィット）
4. 導入実績
5. 比較表（既存代替: 弁護士のみ / 自社人力 vs 自社）
6. 価格
7. FAQ
8. 最終CTA
```

## 期待される必須コピー要素

### FV
- メインキャッチ: 「契約書のチェック工数を、最大70%削減」
- サブコピー: 「AIが見落としがちな条項を瞬時に検出。最終チェックは弁護士へ。」
- 主CTA: 「無料で資料請求」
- 副CTA: 「電話で相談」（営業時間表示）

### 注釈（必須）
- 「最終チェックは弁護士へ」
- 「効果には個人差があります」
- 「※自社調べ・XX社平均・YYYY年実績」
