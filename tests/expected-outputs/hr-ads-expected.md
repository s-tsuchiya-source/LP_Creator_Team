# Expected Output: HR / 求人広告連動LP

## 期待される必須セクション

```
1. FV（職種 × 年収レンジ + 主CTA「応募」+ 副CTA「個別相談」）
2. 「こんな経験ある方を求めています」
3. 仕事内容（具体的）
4. 期待役割と評価軸
5. キャリアパス（5年後）
6. 社員インタビュー
7. 福利厚生・働き方
8. 選考フロー
9. FAQ
10. 最終CTA
```

## 期待される必須実装要素（広告流入特化）

### パフォーマンス
- LCP < 2.5s（広告品質スコア対策）
- CLS < 0.1
- INP < 200ms

### 計測
- GA4 イベント設計
- Google Ads CV（form_submit）
- Meta Pixel Lead
- Yahoo!広告（該当時）
- UTM 取得

## 期待される Emotional CVR 要素

### 自分ごと化
- 広告キーワード × FV のメッセージマッチ
- 「[職種]の経験X年以上の方へ」

### 期待感
- 給与レンジ（実態）
- キャリアパス明示
- 評価制度の運用実績
- 同じ職位の社員インタビュー

### 安心
- 離職率（高すぎなければ明示）
- 評価制度の透明性
- リモートワーク実態

### 不安解消
- 「年収レンジは本当か」→ 評価制度詳述
- 「カルチャーフィット」→ 社員インタビュー
- 「リモート本当に可能か」→ 制度実態数字
- 応募後の流れ明示

### CV直前
- 応募CTA + 個別相談CTA
- 「面接で話を聞くだけ」OK
- スカウトメール受信オプション

## 期待される法務要素

- 男女・年齢制限表現なし
- 「絶対に転職できる」NG
- 「年収XX万円保証」NG
- 不当景品類及び不当表示防止法

## 期待される scorecard

- lp-master: **90点以上**
- emotional-cvr: **85点以上**
- benefit-clarity: **85点以上**
- expectation: **85点以上**
- self-relevance: **85点以上**
- legal-risk: **95点以上**
- final-output（実装品質）: **90点以上**（LCP / CLS / INP 達成）

## NG出力例

- 広告メッセージマッチ崩壊
- LCP > 2.5s
- 計測タグ未実装
- 給与レンジが極端すぎる（誇大）
- 「絶対に転職できる」表現
