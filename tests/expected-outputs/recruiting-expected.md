# Expected Output: 採用LP

## 期待される必須セクション

```
1. FV（ミッション or 仕事魅力 + 社員写真 + カジュアル面談CTA）
2. 「こんな方を求めています」
3. 仕事内容・期待役割
4. 技術スタック
5. 1日の流れ（重要）
6. 社員インタビュー（3名以上）
7. 制度・福利厚生
8. オフィス・働き方
9. 選考フロー（カジュアル面談 → 1次 → 最終）
10. FAQ
11. 最終CTA（カジュアル面談）
```

## 期待される必須コピー要素

- メインキャッチ: ミッション型 or 仕事魅力型
- 主CTA: 「カジュアル面談を予約する」（ローハードル）
- 副CTA: 「メンバーと話す」 or 「採用ピッチ動画」

## 期待される必須数値根拠

- 在籍社員数 / 男女比
- 業界別年収レンジ（実態）
- 評価制度の運用実績
- 「個人差があります」明記

## 期待される必須 Emotional CVR 要素

### 自分ごと化
- FV「[職種]を募集」明示
- 「こんな経験ある方を求めています」

### 期待感（採用LP特有）
- 「自分のスキルが活きる」期待
- 「成長できる」期待（具体的な制度）
- 「キャリアパス」の明示
- 「ここで働く未来」を1日の流れで描く

### 安心
- 社員インタビュー（実在性）
- カルチャー説明
- リモートワーク実態

### 不安解消
- カルチャーフィット不安
- 失敗リスク
- 給与レンジへの不安
- カジュアル面談ハードル下げ

### CV直前
- カジュアル面談（最低ハードル）が主CTA
- 「応募ではないので、お断りも歓迎」
- 「履歴書不要」「事前準備不要」

### CV後
- 面談予約後の流れ
- 当日の進め方

## 期待される scorecard

- lp-master: **90点以上**
- emotional-cvr: **85点以上**
- benefit-clarity: **85点以上**
- expectation: **85点以上**
- self-relevance: **85点以上**
- legal-risk: **95点以上 / Critical Fail なし**

## 期待される法務要素

- 性別・年齢制限表現なし
- 「絶対に活躍できる」NG
- 写真・氏名は社員許諾必須
- 給与レンジは実態と整合

## NG出力例

- 「成長できる」「やりがい」だけの抽象訴求
- フリー素材の人物写真使用
- 1日の流れがない
- カジュアル面談CTAなし
- 「絶対に活躍」「100%評価」表現
- 性別 / 年齢制限表現
