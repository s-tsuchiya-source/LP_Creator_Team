# Benchmark: 採用LP（架空企業）

## テストインプット

```text
LP_Creator_Team の lp-director を起点に、採用LP制作を開始してください。

会社: BlueOcean Tech（架空 / SaaS / 50名 / Series B）
募集: バックエンドエンジニア（3〜5年経験）
CV: カジュアル面談予約
業種: HR / 採用LP

戦略 → 構成 → コピー → デザイン方針 まで出してください。

法務（性別 / 年齢制限表現）に注意してください。
80点未満は差し戻し、90点以上を納品基準としてください。
Emotional CVR scorecards 4種は85点以上必須です。
```

## 検証ポイント

### 1. ペルソナ
- 求職者ペルソナ（30〜35歳エンジニア）が描かれているか
- 業務上 + 感情上のジョブ両方
- 不安・反論リスト

### 2. 法務
- 性別・年齢制限表現なし
- 「絶対に活躍できる」等の保証なし

### 3. 構成
- FV: ミッション or 仕事の魅力
- 「こんな方を求めています」
- 仕事内容・期待役割
- 1日の流れ（重要）
- 社員インタビュー
- 制度・福利厚生
- 選考フロー（カジュアル面談 → 1次 → 最終）
- FAQ
- 最終CTA

### 4. Emotional CVR（採用LP特有）
- 「ここで働く未来」が想像できるか
- 「自分のスキルが活きる」期待
- 「カルチャーが合う」安心
- 「メンバーと働きたい」親近感

### 5. CTA
- カジュアル面談（ローハードル）が主CTA
- 応募（ハイハードル）は副CTA
