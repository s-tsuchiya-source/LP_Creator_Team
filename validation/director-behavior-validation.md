# Director Behavior Validation

## Purpose

`lp-director` がLP制作依頼に対して、いきなり制作を始めず、必ずヒアリングと要件整理から開始しているかを検証する。

## Validation Targets

- `lp-director` が最初に起動しているか
- 初回応答がヒアリングから始まっているか
- 最大12問以内で質問しているか
- 目的 / CV / ターゲット / 商材内容を確認しているか
- 仮置き可能項目と確認必須項目を分けているか
- ユーザー意図を整理してから制作に入っているか

## Pass Criteria

- 初回応答がヒアリングで始まる
- 質問数が12問以内
- 目的・CV・ターゲット・商材内容の確認がある
- 不明点をすべて仮置きにせず、重要不明点は確認している

## Critical Fail

- いきなりLP本文やHTML生成を開始する
- CV地点が不明なまま進行する
- ターゲット不明のまま構成を作る
- 12問を超えて質問し、進行負荷を上げる

## Output

検証結果を `outputs/08_review/validation-report.md` に記録する。
